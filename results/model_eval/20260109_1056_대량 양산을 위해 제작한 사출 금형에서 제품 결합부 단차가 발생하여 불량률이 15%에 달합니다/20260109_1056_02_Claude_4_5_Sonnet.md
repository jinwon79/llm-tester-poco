## 응답 내용
# [기술·제조 전문가 + 법률 분야 보조 관점 통합 답변]

안녕하세요. **제조 기술 및 품질 관리 전문가**로서, 사출 금형의 기술적 책임 소재 규명과 함께 **법률적 대응 전략**을 포함하여 답변드리겠습니다.

---

## 📋 [상황 진단]

### 파악된 핵심 문제
- **불량 유형**: 제품 결합부 단차 발생 (15% 불량률)
- **양측 주장 대립**: 
  - 공장 측: DFM(Design for Manufacturing) 설계 문제
  - 키즈봇 측: 금형 가공 오차
- **비즈니스 임팩트**: 출고 전면 중단 → 매출 손실, 고객 신뢰도 하락 위험

### 추가 필요 정보 (정확한 진단을 위해)
더 정밀한 솔루션 제공을 위해 다음 정보를 확인해주시면 도움이 됩니다:

1. **계약서상 금형 검수 조항**: 초도품 승인(FAI), 치수 보증 범위 명시 여부
2. **3D 설계 도면**: 공차 표기 내역 (±0.05mm 등)
3. **금형 제작 전 DFM 리뷰**: 공장과의 사전 협의 문서 존재 여부
4. **측정 데이터**: CMM(3차원 측정기) 검사 결과 유무
5. **금형 계약 금액 및 납품 조건**: 수정 의무 범위 확인

## 🔍 [핵심 솔루션: 기술적 책임 소재 판별 프로세스]

### 1단계: 객관적 증거 수집 (48시간 내 실행)

| 검증 항목 | 실행 방법 | 책임 판별 기준 |
|---------|---------|-------------|
| **설계 도면 공차 확인** | 2D 도면의 결합부 치수 공차 재검토 | 공차가 ±0.05mm 이하로 과도하게 타이트 → 설계 문제 의심 |
| **금형 가공 정밀도 측정** | 공장에 CMM 측정 성적서 요구 | 측정값이 도면 공차 범위 이탈 → 금형 가공 문제 |
| **초도품(T1) 승인 기록** | 양산 전 샘플 검수 당시 이슈 여부 확인 | T1에서 단차 미발견 → 양산 중 금형 변형/마모 가능성 |
| **사출 조건 검증** | 사출 온도, 압력, 냉각 시간 표준 대비 점검 | 조건 이탈 시 → 사출 파라미터 조정 필요 (양측 책임 공유) |

**즉시 실행 사항**:
- 공장에 "CMM 측정 성적서 제출 + 불량품 10개 샘플 송부" 공문 발송 (내용증명 권장)
- 자사 설계 도면을 외부 DFM 전문업체(예: ㈜디에스테크, 유니온테크 등)에 의뢰하여 **제3자 검증** 진행

---

### 2단계: 책임 소재별 대응 시나리오

#### 시나리오 A: 금형 가공 오차가 주 원인인 경우

**근거 확보 방법**:
- CMM 측정 결과 ≥ 3개 이상 부위에서 공차 이탈
- 초도품 승인 시에는 문제 없었으나 양산 중 발생 (금형 마모/변형)

**협상 전략**:
```
1. 계약서상 '치수 보증 조항' 명시 확인
   → 있을 경우: 무상 수정 요구
   → 없을 경우: 하자 담보 책임(민법 제580조) 원용

2. 수정 방식 제안
   - Insert 교체: 결합부만 부분 수정 (비용 효율적)
   - 전면 재가공: 코어/캐비티 재작업 (시간 소요)

3. 비용 분담 협상안
   - 공장 부담 80% + 키즈봇 20% (신속 처리 조건)
   - 향후 금형 유지보수 계약 체결로 장기 관계 유지
```

**법적 백업**:
- 내용증명 발송: "계약 불이행 → 7일 내 수정 조치 요구"
- 하자보수 청구권 행사: 민법 제667조(수급인의 담보책임)

---

#### 시나리오 B: 설계(DFM) 문제가 주 원인인 경우

**근거 확보 방법**:
- 외부 DFM 검증 결과, 결합부 구조가 사출 성형에 부적합 판정
- 언더컷 과다, 게이트 위치 부적절, 냉각 불균형 등 확인

**즉시 대응**:
```
1. 설계 수정안 작성 (외주 업체 활용)
   - 결합부 스냅핏 구조 변경 또는 단차 허용 범위 확대
   - 리브 두께 조정, 게이트 위치 변경 등

2. 공장과 협의
   - "설계 변경 범위 내 금형 수정" 협의 (일부 비용 분담)
   - 기존 금형 살리며 Insert만 교체하는 방향 제안

3. 내부 프로세스 개선
   - 향후 금형 제작 전 반드시 DFM 리뷰 단계 추가
   - 공장과 Pre-T1 단계 신설 (시제 금형 검증)
```

---

### 3단계: 협상력 확보를 위한 추가 전략

#### A. 제3자 중재 활용
- **한국금형공업협동조합**: 금형 분쟁 조정 서비스 제공
- **중소벤처기업진흥공단**: 기술 분쟁 조정 지원 (무료)
- **외부 품질 인증기관**: KTR, KTL 등에 의뢰하여 공신력 있는 측정 리포트 확보

#### B. 대체 공급망 확보
- 병행하여 2~3개 금형 업체에 견적 요청 (협상 레버리지 확보)
- "수정 불가 시 타 업체 이관" 카드 활용 (단, 관계 악화 주의)

#### C. 비용 최소화 방안
- **정부 지원 활용**: 
  - 소재·부품 품질 개선 지원사업 (중진공)
  - R&D 긴급 지원 프로그램 신청 (금형 수정 비용 일부 지원)

## 📊 [Step-by-Step 실행 가이드]

### 🚨 즉시 실행 (D+1~2일)
1. ✅ 공장에 CMM 측정 성적서 + 불량 샘플 요청 (공문 발송)
2. ✅ 자사 설계 도면 외부 DFM 검증 의뢰
3. ✅ 계약서 및 초도품 승인 기록 재검토
4. ✅ 변호사 자문: 계약서상 하자 보수 조항 확인

### 🔧 단기 조치 (D+3~7일)
5. ✅ 측정 데이터 기반 책임 소재 1차 판별
6. ✅ 공장과 대면 미팅: 협상안 제시
   - 금형 문제 시: 무상 수정 요구 + 일정 합의
   - 설계 문제 시: 수정 범위 최소화 방안 협의
7. ✅ 제3자 중재 기관 연락 (협상 난항 시 대비)

### 🏭 중장기 조치 (D+8~30일)
8. ✅ 금형 수정 완료 후 재검증 (CMM + 양산 시험)
9. ✅ 내부 프로세스 개선
   - DFM 체크리스트 작성
   - 금형 업체 선정 기준 강화 (ISO 9001 필수)
10. ✅ 향후 재발 방지 조항 계약서 반영
