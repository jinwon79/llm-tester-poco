## 💡 핵심 솔루션: 저가법 적용 원칙

### ✅ **결론부터 말씀드리면**

| 재고 유형 | 평가 기준 | 귀사 적용 방법 |
|---------|---------|-------------|
| **완제품·상품** | **순실현가능가치(NRV)** | 판매예상가 - 추가완성원가 - 판매비용 |
| **원재료** | **①원칙: 현행대체원가**<br>②예외: 완제품 NRV 연동 | ⚠️ **완제품 NRV < 원가일 경우만 연동 평가** |

## 📘 회계기준 상세 해석

### 1️⃣ **K-IFRS 제1002호 (재고자산) 핵심 조항**

```
제28조: 원재료 및 기타 소모품이
(1) 정상적인 시장가격(현행대체원가)으로 취득 가능하면 → 원가 이하로 평가 불필요
(2) 완제품 제조 후 판매 시 순실현가능가치 미달이 예상되면 → 원재료도 NRV 평가
```

### 2️⃣ **귀사 적용 로직**

```
[STEP 1] 완제품 수익성 테스트
완제품 판매예상가 - 추가가공원가 - 판매비용 = 완제품 NRV

[STEP 2] 원재료 평가 기준 결정
IF 완제품 NRV ≥ 완제품 원가 
   → 원재료는 '현행대체원가'로 평가 (시장가 하락해도 평가손실 X)

IF 완제품 NRV < 완제품 원가
   → 원재료도 '완제품 NRV 연동' 평가 (평가손실 인식 ○)
```
