**추가로 필요한 정보:**

- 원재료 재고 회전율 (월평균 소진 기간)
- 완제품 판매가격 결정 방식 (원가연동형 / 고정가격 계약)
- 현재 원재료 장부금액 대비 현행대체원가 하락률(%)
- 기말 재고 규모 (금액 기준)

## [핵심 솔루션: 저가법 적용 기준]

### ✅ **결론부터 말씀드리면**
> **"원재료는 원칙적으로 완제품의 순실현가능가치와 연동하여 평가해야 하며, 현행대체원가를 그대로 쓸 수 없습니다."**

### 📋 **K-IFRS 제1002호 재고자산 기준 해석**

| 구분 | 평가 기준 | 적용 방법 |
|------|----------|----------|
| **완제품/상품** | 순실현가능가치(NRV) | 추정 판매가격 - 추정 판매비용 |
| **원재료** | **①** 현행대체원가<br>**②** 완제품 NRV 연동 | 완제품이 원가 이상으로 팔릴 경우 → ① 적용<br>완제품이 원가 미만으로 팔릴 경우 → ② 적용 |

### 🔍 **귀사 사례 적용 로직**

```
[STEP 1] 완제품 순실현가능가치(NRV) 산출
완제품 예상 판매가 100,000원
- 예상 판매비용 (5,000원)
= NRV 95,000원

[STEP 2] 완제품 원가 구성 확인
원재료비 60,000원
가공비 40,000원
= 총 원가 100,000원

[STEP 3] 원재료 평가 판단
∵ 완제품 NRV(95,000) < 완제품 원가(100,000)
∴ 원재료도 평가손실 인식 필요

[STEP 4] 원재료 장부금액 조정
원재료 현행 장부가 60,000원
→ 조정 후 가치 = 95,000 - 40,000(가공비) = 55,000원
→ **평가손실 5,000원 인식**
```

## [리스크 체크리스트]

### ⚠️ **주의해야 할 회계/세무 리스크**

**1. 과소평가 리스크**
- ❌ 잘못된 사례: 원재료 현행대체원가만 보고 평가손실 과다 계상
- ✅ 올바른 방법: 완제품이 정상 마진으로 판매 가능하면 원재료 평가손실 불필요

**2. 세무조정 이슈**
```
법인세법상 재고자산 저가법 인정 요건:
- 파손/진부화 등 '물리적 손상' 또는
- 시가 하락이 '일시적이지 않은 경우'에 한정

⚠️ 단순 시황 변동은 세무상 손금 불산입 가능성 있음
→ 회계상 평가손실 계상 후, 세무조정(유보) 필요할 수 있음
```

**3. 외부감사 대응**
- 재고실사 시 회계법인에 평가 근거자료 제출 필수
- 판매가격 산정의 합리성 입증 (견적서, 계약서 등)
