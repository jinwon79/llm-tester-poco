## 🔍 상황 진단

**(주)스틸솔루션**의 현재 상황을 다음과 같이 파악했습니다:

| 항목 | 현황 | 회계적 쟁점 |
|------|------|------------|
| **업종** | 특수강 가공 (제조업) | 원재료 → 완제품 전환 과정 존재 |
| **재고 구성** | 원재료(특수강) + 재공품 + 완제품 | 각 단계별 저가법 적용 기준 상이 |

| **주요 고객** | 자동차/중장비 협력사 | 완제품 판매가 하락 압력 존재 |

## 💡 핵심 솔루션: 저가법 적용 원칙

### **결론부터 말씀드리면:**
> **원재료는 '현행대체원가'를, 완제품은 '순실현가능가치'를 각각 적용하되,  
> 원재료가 완제품 제조에 사용될 경우 완제품의 판매 가능성을 고려해 평가**해야 합니다.

---

### 📘 K-IFRS/일반기업회계기준 기준 해석

#### **1️⃣ 원재료 평가 원칙 (핵심)**

```
┌────────────────────────────────────────┐
│ 원재료 저가법 적용 기준 판단 흐름도      │
└────────────────────────────────────────┘

[1단계] 완제품 판매가 확인
   ↓
완제품 판매가 ≥ 제조원가?
   ↓
   YES → 원재료는 취득원가 그대로 평가 (손상 없음)
   NO  → [2단계]로 이동
   
[2단계] 원재료 현행대체원가 확인
   ↓
현행대체원가 < 취득원가?
   ↓
   YES → 현행대체원가로 평가 (평가손실 인식)
   NO  → 취득원가 유지
```

#### **2️⃣ 법률적 근거**

| 기준 | 조항 | 핵심 내용 |
|------|------|----------|
| **K-IFRS 1002호** | 문단 32 | "원재료 등 **제조에 사용되는 재고자산**은 완제품 판매 시 순실현가능가치 이상으로 회수 가능하면 **취득원가 이하로 평가하지 않음**" |
| **일반기업회계기준** | 10.14 | "원재료를 가공한 완제품의 순실현가능가치가 원가 이상이면 원재료는 손상되지 않은 것으로 봄" |

## 🛠 Step-by-Step 실행 가이드

### **STEP 1: 완제품 판매 가능성 테스트 (48시간 내)**

```
[필수 확인 사항]
☑ 최근 3개월 평균 판매단가 산출
☑ 주요 거래처 견적 갱신 여부 확인
☑ 기말 시점 수주잔고(Order Backlog) 검토
```

**📝 실무 작업 시트**
```
완제품 A 기준:
- 현재 판매단가: ______원/kg
- 직접재료비: ______원/kg (원재료 투입량 × 현재 단가)
- 가공비(노무비+경비): ______원/kg
- 판매비: ______원/kg
─────────────────────
= 총 제조원가: ______원/kg

→ 판매단가 ≥ 총 제조원가?
   YES: 원재료 평가손실 불필요
   NO: STEP 2로 이동
```

---

### **STEP 2: 원재료 현행대체원가 확인 (1주일 내)**

#### **현행대체원가 산정 방법**
```
옵션 1: 기말 시점 시장 시세 (철강협회 고시가 등)
옵션 2: 주요 공급업체 최근 견적가 (3사 이상)
옵션 3: 수입 원자재의 경우 CIF 가격 + 관세
```

**⚠️ 주의사항:**
- 단순 "시세 하락"만으로 평가손실 인식 금지
- 실제 **구매 가능한 가격**(재주문 시 적용 가격)을 기준으로 해야 함
- 일시적 시장 교란(예: 펜데믹, 전쟁)은 기업특유 상황 고려

---

### **STEP 3: 회계처리 시나리오별 대응 (결산 전)**

#### **시나리오 A: 완제품 판매가 유지 (가장 흔한 케이스)**
```
[회계처리]
→ 평가손실 인식 안 함
→ 원재료는 취득원가 그대로 재무제표 표시

[근거]
K-IFRS 1002.32: "원재료가 정상 판매가로 전환 가능하면 
순실현가능가치 ≥ 원가로 간주"
```

#### **시나리오 B: 완제품 판매가 하락 (귀사 상황 추정)**
```
[회계처리 단계]
1. 완제품별 순실현가능가치(NRV) 산정:
   NRV = 예상판매가 - 예상 판매비용 - 완성까지 추가원가

2. 원재료 저가법 적용:
   평가손실 = (취득원가 - 현행대체원가) × 재고수량
   
   ※ 단, 완제품 NRV가 여전히 제조원가보다 높으면
      원재료 평가손실 불인식

3. 재무제표 표시:
   (차) 재고자산평가손실 XXX  (대) 원재료 XXX
```

#### **시나리오 C: 원재료 일부는 판매, 일부는 제조**
```
[분리 평가 필요]
• 직접 판매용 원재료: 순실현가능가치로 평가
• 제조 투입용 원재료: 완제품 NRV 기준 평가
```
