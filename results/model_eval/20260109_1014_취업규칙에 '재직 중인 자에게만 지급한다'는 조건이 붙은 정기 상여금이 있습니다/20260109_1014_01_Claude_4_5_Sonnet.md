## 🔍 핵심 솔루션: 최신 판례 기준 적용

### 1️⃣ **"재직자 지급 조건" 상여금의 통상임금 해당 여부**

#### ✅ **대법원 전원합의체 판결(2013.12.18. 선고 2012다89399)의 핵심 기준**

| 판단 요소 | 통상임금 포함 조건 | 귀사 상황 분석 |
|---------|-----------------|--------------|
| **정기성** | 일정한 주기로 지급 | ✓ 정기 상여금 → 해당 |
| **일률성** | 모든 근로자에게 동일 조건 | ✓ 재직자 전원 → 해당 |
| **고정성** | 근로 제공 시 당연 지급 | **⚠️ 핵심 쟁점!** |

#### 🚨 **최신 판례상 "재직 중" 조건의 효력**

```
대법원 판례 요지 (2012다89399):
"지급일에 재직 중인 자에게만 지급한다는 조건은 
근로 제공과 무관한 '사후적·우연적 조건'에 해당하여 
고정성을 결여하지 않는다"

→ 즉, 재직자 조건만으로는 통상임금에서 제외 불가!
```

**결론:** 귀사의 정기 상여금은 **통상임금에 포함될 가능성이 매우 높습니다.**

### 2️⃣ **소급 적용 리스크 분석**

#### 📊 **3년 소급 청구 시 추정 재무 영향**

```
[가정: 생산직 40명, 월 평균 기본급 300만원, 상여금 400%]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 통상임금 증가분 계산
   - 기존 통상임금: 300만원
   - 상여금 월할 산입: 300만원 × 400% ÷ 12개월 = 100만원
   - 신규 통상임금: 400만원 (33% 증가)

2. 연장수당 차액 (월 평균 50시간 가정)
   - 기존: 300만원 ÷ 209시간 × 1.5배 × 50시간 = 107.6만원
   - 신규: 400만원 ÷ 209시간 × 1.5배 × 50시간 = 143.5만원
   - 차액: 35.9만원/인·월

3. 3년 소급 총액
   - 35.9만원 × 40명 × 36개월 = 약 5.16억원
   - 퇴직금 추가 산정분: 약 0.8억원
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   총 추정 부담: 약 6억원 (이자 및 지연가산금 미포함)
```

#### ⚖️ **소급 적용 제한 가능성 (신의칙 항변)**

대법원은 **과도한 소급적용의 부담**을 인정하여 **신의칙 항변**을 허용했습니다:

| 신의칙 적용 요건 | 입증 책임 | 귀사 적용 가능성 |
|---------------|---------|---------------|
| ① 사용자의 선의·무과실 | 사용자 | △ 2013년 판결 이후 모르긴 어려움 |
| ② 노사 합의 존재 | 사용자 | ✓ 단체협약·취업규칙 근거 필요 |
| ③ 기업의 지급 불능 | 사용자 | △ 재무제표 기반 입증 필요 |
| ④ 근로자의 소급 기대 부재 | 사용자 | ✗ 노조 요구로 기대 명확 |

**현실적 판단:** 노조가 명시적으로 요구한 상황에서 **신의칙 항변 성공 가능성은 낮습니다** (약 30% 이하).

## 📌 Step-by-Step 실행 가이드

### 🔴 **즉시 조치 (1주 이내)**

```
1. 긴급 리스크 정량화
   □ 최근 3년간 연장·야간·휴일 근로 시간 전수 조사
   □ 회계법인과 협력하여 정확한 소급액 산출
   □ 현금흐름 기준 최대 지급 가능액 시뮬레이션

2. 노무법인 긴급 자문
   □ 취업규칙 및 단체협약의 법적 하자 검토
   □ 신의칙 항변 가능성 정밀 분석
   □ 소송 시 승소 확률 및 기간 예측
```

### 🟡 **단기 대응 (1개월 이내)**

```
3. 노사 협상 전략 수립
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   [제안 1] 향후 적용 방식 (Forward-Looking)
   - 통상임금 산입을 인정하되 소급 적용은 배제
   - 대신 향후 임금체계 개편 시 반영 (예: 2024년 1월부터)
   
   [제안 2] 분할 지급 합의
   - 소급액의 30~50%만 인정
   - 3~5년 분할 지급 (이자 없이)
   
   [제안 3] 임금체계 재설계
   - 상여금을 기본급에 통합 (총액 불변)
   - 대신 성과급 체계 신설 (변동 항목화)
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

4. 법적 방어 준비
   □ 과거 노사 간 통상임금 관련 합의 문서 확보
   □ 업계 관행 및 타사 사례 수집 (신의칙 입증 자료)
   □ 필요시 "임금채권 소멸시효(3년)" 적용 검토
```

### 🟢 **중장기 전략 (3~6개월)**

```
5. 임금체계 개편 (통상임금 리스크 최소화)
   
   [개편 방향]
   ┌─────────────────────────────────────┐
   │ 기존: 기본급 60% + 상여금 40%        │
   │ 개편: 기본급 80% + 성과급 20%        │
   │       (성과급은 KPI 연동, 비고정 항목)│
   └─────────────────────────────────────┘
   
   ※ 주의: 취업규칙 불이익 변경 시 근로자 동의(과반수 또는 노조) 필수

6. 연장근로 관리 시스템 강화
   □ 스마트 근태관리 시스템 도입 (자동 집계)
   □ 연장근로 사전승인제 시행
   □ 탄력근로제 또는 선택근로제 검토
```
