# 📋 노무/인사 전문가로서 답변드리겠습니다

귀사의 정기 상여금 통상임금 산입 이슈는 **대법원 2013다89399 전원합의체 판결(2013.12.18)** 이후 가장 민감한 노무 쟁점 중 하나입니다. 제조업 특성상 연장·야간·휴일근로가 빈번한 귀사의 경우, 통상임금 재산정 시 재무적 영향이 클 것으로 예상됩니다.

## 🔍 상황 진단

### 파악된 정보
- **업종**: 자동차 부품 제조 (연장근로 다발 업종)
- **인원**: 생산직 40명 + 품질관리 5명 = 총 45명
- **쟁점**: 재직 조건부 정기 상여금의 통상임금 포함 여부
- **시급성**: 노조의 협상 요구 + 3년 소급 리스크

### 추가 필요 정보
더 정확한 리스크 산정을 위해 다음 정보를 확인해주시면 맞춤형 대응 전략 수립이 가능합니다:

1. **상여금 구조**
   - 정기 상여금 지급 시기 (연 몇 회?)
   - 1인당 평균 지급액
   - 기본급 대비 비율 (예: 기본급의 500%)

2. **근로 패턴**
   - 월평균 연장근로 시간
   - 주말/야간 근무 빈도
   - 교대제 운영 여부

3. **취업규칙 상세**
   - 정확한 문구 ("퇴직자 불지급" 또는 "재직자 전원 지급")
   - 과거 실제 지급 사례 (중도 퇴사자 지급 여부)

#### ⚖️ 판례의 핵심 요지

> 단순히 '재직자에게만 지급한다'는 조건만으로는 통상임금에서 배제할 수 없다."

### 2. 귀사 상여금의 통상임금 포함 여부 자가 진단

```
✅ 다음 질문에 "예"가 많을수록 포함 가능성 ↑

□ 매년 동일한 시기에 지급하는가? (예: 설날·추석·7월·12월)
□ 모든 근로자에게 동일한 산식으로 지급하는가? (직급 무관)
□ 결근·징계 등 없으면 무조건 지급하는가?
□ 과거 5년간 지급일에 재직한 사람은 전원 수령했는가?
□ 기본급과 연동된 고정 비율인가? (예: 기본급의 400%)
```

**판단**: 4개 이상 "예" → **통상임금 포함 가능성 80% 이상**

## 📌 Step-by-Step 실행 가이드

### 🚨 즉시 실행 (이번 주 내)

#### 1단계: 증거 확보 및 현황 파악
```
✓ 최근 5년간 상여금 지급 대장 확인
  - 지급일 기준 재직자 전원 수령 여부
  - 중도 퇴사자 지급 사례 유무
  
✓ 취업규칙 및 단체협약 정밀 검토
  - "재직 중인 자"의 정확한 문구
  - 과거 개정 이력
  
✓ 실제 근로시간 기록 확인
  - 타임카드/전자출퇴근 데이터 보존 상태
  - 연장·야간·휴일근로 집계 정확성
```

#### 2단계: 법률 자문 (노무법인)
```
✓ 체크리스트 지참 방문:
  ① 취업규칙 원본
  ② 5년간 상여금 지급 내역
  ③ 노조 요구안 문서
  ④ 현행 임금 구조표
  
✓ 자문 필수 질문:
  - 현재 규정으로 방어 가능성은?
  - 소급 시 최소·최대 시나리오는?
  - 노조 협상 시 유리한 카드는?
```

---

### 📊 중기 대응 (1~3개월)

#### 3단계: 협상 시나리오 설계

**Option A: 통상임금 인정 + 소급 제한 협상**
- 향후 상여금 → 통상임금 포함 인정
- 과거 3년 → 일시금 합의 (예: 50% 할인)
- **장점**: 법적 리스크 최소화
- **단점**: 단기 재무 부담 ⬆️

**Option B: 임금체계 재편**
- 상여금을 기본급으로 통합 (총액 동결)
- 통상임금은 ⬆️, 하지만 총 인건비는 동일
- **장점**: 장기적 안정성 확보
- **단점**: 노조 설득 난이도 ⬆️

**Option C: 단계적 개선안**
- 1년차: 향후 발생분만 적용
- 2~3년차: 점진적 소급 (연 1억씩)
- **장점**: 현금흐름 관리 용이
- **단점**: 노조 반발 가능성

#### 4단계: 취업규칙 개정 (전문가 검토 필수)
```
❌ 위험한 문구:
"상여금은 재직 중인 자에게만 지급한다"

✅ 개선 방향:
"상여금은 소정근로시간에 대한 대가가 아니며,
 회사의 경영성과 및 개인의 근속·성과에 따라
 지급 여부와 금액이 결정되는 특별급여이다"
```

---

### 🔐 장기 안정화 (6개월~1년)

#### 5단계: 임금피크제 or 성과급 전환 검토
- 정기 상여금 → 성과 연동 변동급 전환
- 통상임금 고정화 억제
- 단, 근로기준법 제94조(취업규칙 불이익 변경) 주의

#### 6단계: 시스템 고도화
- 통합 급여 관리 시스템 도입
- 실시간 통상임금 자동 계산
- 연장근로 사전 승인제 (예산 통제)
