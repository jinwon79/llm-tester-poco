# Claude 4.5 Sonnet  응답 (2/3회차)

## 응답 내용
# 노무/인사 전문가로서 답변드리겠습니다

귀사의 상황은 **통상임금 산정 기준 변화에 따른 임금체계 전면 재검토가 필요한 중대한 국면**으로 판단됩니다. 제조업 특성상 연장·야간·휴일근로가 빈번하므로, 통상임금 범위 확대 시 재무적 영향이 매우 클 것으로 예상됩니다.

---

## 📋 상황 진단

### 파악된 현황
- **쟁점 조항**: 취업규칙상 "재직 중인 자에게만 지급" 조건부 정기 상여금
- **노조 요구**: 해당 상여금의 통상임금 산입 및 과거 3년분 소급 지급
- **기업 규모**: 생산직 중심 45명 (노조 조직 가능 규모)
- **리스크 노출도**: 제조업 특성상 초과근로 다발 → 통상임금 확대 시 재무 부담 급증

### 추가 확인 필요 사항
더 정확한 컨설팅을 위해 다음 정보를 제공해주시면 맞춤형 솔루션이 가능합니다:
1. **월 평균 연장·야간·휴일 근무시간** (최근 1년 기준)
2. **정기 상여금 지급 규모** (월급 대비 비율 또는 연간 총액)
3. **취업규칙 제·개정 이력** (특히 상여금 조항 변경 시점)
4. **현재 노조 교섭 단계** (요구서 접수 / 본교섭 / 조정신청 등)

|------|------------------------|----------------------------|
| **"재직자 지급" 조건의 효력** | 통상임금 제외 요인으로 인정 | **원칙적으로 통상임금에 포함** |
| **핵심 판단 기준** | 형식적 조건 존재 여부 | ① 정기성 ② 일률성 ③ 고정성 실질 판단 |
| **귀사 사례 적용** | 조건부 → 제외 가능 | 매월/분기별 정기 지급 시 **포함 가능성 높음** |

#### 판례 핵심 요지
> "근로의 대가로서 정기적·일률적·고정적으로 지급되는 임금은, 지급 조건에 '재직 중인 자'라는 제한이 있더라도 **실질적으로 모든 근로자에게 지급되는 경우** 통상임금에 해당한다." (대법원 2012다89399 전원합의체)

**귀사 상여금 판단 체크리스트**:

- [ ] 전 직원 동일 지급률 적용? → 일률성 확인 필요
- [ ] 실제 퇴사자 외 지급률 100%? → **고정성 인정 가능**

⚠️ **결론**: 귀사의 경우 **형식적 조건에도 불구하고 통상임금 포함 가능성이 매우 높습니다.**

## 🎯 Step-by-Step 실행 가이드

### PHASE 1: 긴급 대응 (즉시 ~ 2주)

#### 1-1. 법률 검토 및 리스크 정량화
```
[실행 체크리스트]
□ 노무법인 자문 계약 (임금체계 전문 변호사 필수)
□ 취업규칙·단체협약 전수 조사
  - 상여금 지급 조건·규모·지급률 확인
  - 과거 3년 실제 지급 내역 추출
□ 재무 시뮬레이션
  - 시나리오 A: 통상임금 산입 + 3년 소급
  - 시나리오 B: 향후 임금체계 변경 시 추가 부담
```

**즉시 연락처**:
- 한국노동법학회 추천 전문 노무법인
- 고용노동부 임금체계 개선 컨설팅 (무료) ☎ 1350

#### 1-2. 노조 대응 전략 수립
```
[교섭 원칙]
1. 시간 확보: "정밀한 법리 검토 후 성실 교섭" 입장 표명
2. 단계적 제안: 일시 소급 불가 시 분할 지급 방안 준비
3. 대안 제시: 복리후생 확대 등 총액 임금 재설계 제안
```

⚠️ **주의**: 일방적 거부는 부당노동행위 소지 → 반드시 **성실 교섭 의무** 이행

---

### PHASE 2: 중기 대책 (1~3개월)

#### 2-1. 임금체계 재설계 (통상임금 리스크 최소화)

| 현행 구조 | 문제점 | 개선 방안 |
|----------|--------|----------|
| 정기 상여금 (재직 조건부) | 통상임금 포함 가능 | ① **성과상여금 전환** (변동성 확보) |
| | | ② **복리후생비 전환** (실비 보상) |
| 연장근로 빈번 | 통상임금 증가 시 비용 급증 | ③ **근로시간 단축** (주 40시간 준수) |

**추천 모델: "통합 기본급 + 성과급 분리"**
```
[개편 예시]
기존: 기본급 200만원 + 정기상여 100만원
     → 통상임금 300만원 (상여금 포함 시)

개선: 기본급 250만원 + 성과상여 최대 100만원
     → 통상임금 250만원 (성과급 제외)
     → 장기적 인건비 안정화
```

#### 2-2. 근로시간 관리 강화
- **연장근로 사전 승인제** 도입
- **탄력적/선택적 근로시간제** 검토 (노사 합의 필수)
- **생산 공정 자동화** 투자 (스마트공장 지원사업 연계)

**활용 가능 정부 지원**:
- 고용부 '임금체계 개편 컨설팅' (진단·설계 무료)
- 중기부 '스마트공장 구축 지원' (최대 1억원)

---

### PHASE 3: 장기 전략 (3~12개월)

#### 3-1. 노사 합의 도출
```
[협상 로드맵]
1단계: 통상임금 산입 인정 + 소급 범위 협의
2단계: 분할 지급 조건 합의 (예: 3년 → 5년 분할)
3단계: 신규 임금체계 이행 협약 체결
```

**Win-Win 제안 예시**:
- 노조: 통상임금 소급분 50% 즉시 지급 + 잔여 분할 지급
- 회사: 향후 임금인상률 조정 + 성과급제 도입 수용

#### 3-2. 재무 안정화 조치
- **소급 지급 재원 확보**: 운영자금 대출 (정책금융 활용)
- **원가 절감**: 협력사 단가 재협상, 불필요 경비 축소
- **매출 확대**: 신규 수주 확보, 단가 인상 추진
