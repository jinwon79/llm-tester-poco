## 🔧 [핵심 DFM 수정 체크리스트]

### **1. 기구 설계 (Mechanical Design)**

| 항목 | 프로토타입 방식 | DFM 수정 방향 |
|------|----------------|---------------|
| **언더컷(Undercut)** | 3D 프린팅으로 자유 형상 | → 금형 분할선 고려, 슬라이드 코어 최소화 |
| **벽 두께** | 불규칙 (0.5~3mm 혼재) | → 균일화 (1.2~1.5mm), 두께 차이 30% 이내 |
| **리브/보스 구조** | 없거나 과도하게 두꺼움 | → 벽 두께의 50~60% 수준으로 재설계 |
| **조립 방식** | 접착제, 나사 다수 | → 스냅핏(Snap-fit) 또는 초음파 용착 검토 |
| **Draft Angle** | 없음 (수직면) | → 1~3도 경사 필수 (이형성 확보) |

**🎯 즉시 실행 항목:**
- 금형 업체에 현재 3D 파일 전달 → **"Draft Angle 및 언더컷 검토 의견서"** 요청
- 기구 설계자와 함께 **파팅라인(Parting Line) 위치 재설정**


### **2. PCB 및 전기 설계**

| 이슈 | 원인 | 해결 방안 |
|------|------|----------|
| **배터리 교체 방식** | 분해 필요 | → 도어 방식 또는 탈착형 설계 (KC 인증 시 중요) |
| **센서 배치** | 외벽과 간섭 | → 공기 유입구와 센서 위치 재배치 (측정 정확도 확보) |
| **커넥터 개수** | 과다 (원가 상승) | → FPC(Flexible PCB)로 통합 또는 B2B 커넥터 사용 |
| **방수/방진** | 미고려 | → IP 등급 설정 (최소 IP54 권장) 후 실링 추가 |

**🎯 즉시 실행 항목:**
- PCB 업체에 **"Assembly Drawing + BOM"** 제출 → 실장 가능 여부 확인
- 센서 공급사에 **"양산 시 납기 및 MOQ"** 확인 (Dust Sensor는 특히 리드타임 길음)


### **3. 원가 구조 재설계**

**일반적인 웨어러블 제조원가 구조 (참고)**
```
부품비(BOM): 50~60%
  - 센서(미세먼지): 15~20%
  - 배터리: 8~12%
  - PCB PCBA: 20~25%
금형비 상각: 10~15% (초도 3,000개 기준)
조립/검사: 5~10%
패키징: 3~5%
불량률: 5~8% (초도 양산)
```

**원가 절감 포인트:**
1. **금형 단순화**: 2~3 Cavity → 1~2 Cavity로 축소 (초도 물량 적을 경우)
2. **범용 부품 사용**: 특수 나사/특수 배터리 → KS 규격 부품으로 변경
3. **후공정 최소화**: 도색/인쇄 → 재료 자체 색상 또는 IMD 필름 적용

## 📅 [Step-by-Step 실행 가이드]

### **Phase 1: 설계 수정 (2~3주)**
**Week 1-2:**
- [ ] 금형 업체 3곳 이상 방문 → 견적 + 기술 컨설팅 받기
- [ ] 기구 설계 DFM 체크리스트 작성 (위 표 기준)
- [ ] PCB 업체와 미팅 → "Test Assembly" 진행

**Week 3:**
- [ ] 수정된 3D 파일 완성
- [ ] 시뮬레이션: Moldflow 분석 (금형 업체 협조) → 충진 불량 예측
- [ ] 원가표 재작성 (목표 제조원가 vs 현실 갭 분석)

### **Phase 2: 시제 금형 제작 (4~6주)**
- [ ] 알루미늄 간이 금형 제작 (정식 금형 전 검증용)
- [ ] 100~300개 테스트 양산 → 불량률/조립성 체크
- [ ] **중요**: 이 단계에서 KC 인증 샘플 확보 필수

### **Phase 3: 양산 금형 및 파일럿 런 (6~8주)**
- [ ] 정식 금형 발주 (수정 여지 최소화된 설계로)
- [ ] 초도 1,000~3,000개 양산
- [ ] 품질 검수 프로세스 확립 (AQL 기준 설정)
