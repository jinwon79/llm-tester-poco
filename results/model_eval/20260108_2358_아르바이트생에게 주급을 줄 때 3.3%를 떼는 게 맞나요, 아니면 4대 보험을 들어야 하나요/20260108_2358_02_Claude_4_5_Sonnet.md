커피매치처럼 **단기·파트타임 인력 중심의 매칭 사업**에서는 근로 형태에 따라 급여 처리 방식이 달라집니다. 3.3% 원천징수와 4대 보험 가입 여부는 **근로시간, 고용 형태, 계약 구조**에 따라 법적으로 명확히 구분됩니다.

## 📋 [상황 진단]

현재 커피매치의 **핵심 이슈**:
- ✅ 주 15시간 미만 근무자와 초과 근무자가 혼재
- ✅ 고용 형태(근로자 vs 프리랜서)가 명확히 분리되지 않음
- ✅ 급여 처리 방식(3.3% vs 4대 보험) 기준이 불명확

**더 정확한 컨설팅을 위해 추가로 필요한 정보**:
1. 바리스타와의 계약 형태 (근로계약서 vs 용역계약서)
2. 실제 주당 평균 근무시간 및 근무 패턴
3. 업무 지시 및 관리 방식 (출퇴근 관리 여부, 업무 재량권)

## 💡 [핵심 솔루션: 3.3% vs 4대 보험 판단 기준]

### ① **3.3% 원천징수 (프리랜서/사업소득자)**

| 적용 대상 | 조건 |
|---------|------|
| 계약 형태 | **용역계약서** (인력 제공 형태) |
| 근로 형태 | 독립적 업무 수행, 시간·장소 자율적 |
| 근무 시간 | 주 15시간 미만 또는 1개월 미만 초단기 |
| 지휘·감독 | 없음 (결과물만 요구) |

**적용 방법**:
- 지급액의 **3.3%** (소득세 3% + 지방소득세 0.3%)를 원천징수
- 5월에 **사업소득 지급명세서** 제출 (홈택스)
- 바리스타는 이듬해 5월 **종합소득세 신고** 필요

---

### ② **4대 보험 가입 (근로자)**

| 적용 대상 | 조건 |
|---------|------|
| 계약 형태 | **근로계약서** (고용 관계) |
| 근로 형태 | 출퇴근 시간 지정, 업무 지시 존재 |
| 근무 시간 | **주 15시간 이상** 또는 1개월 이상 지속 근무 |
| 지휘·감독 | 있음 (근무 시간, 장소, 방법 통제) |

**적용 방법**:
- **국민연금, 건강보험, 고용보험, 산재보험** 가입 의무
- 주 15시간 미만자는 **고용보험·산재보험만** 가입 (건보·국민연금 제외)
- 급여 지급 시 4대 보험료를 **회사와 근로자가 분담** (산재는 회사 100%)

## 📌 [커피매치 상황별 실무 적용 가이드]

### **Case 1: 주 15시간 미만 단기 바리스타 (1일 4시간, 주 2회)**
```
✅ 원칙: 근로자로 분류 (근로기준법 적용)
✅ 4대 보험: 고용보험 + 산재보험만 가입 (건보·국민연금 제외)
✅ 급여 처리:
   - 근로소득으로 간주
   - 간이세액표에 따라 소득세 원천징수 (보통 0원 또는 소액)
   - 근로소득 지급명세서 제출 (연말정산 대상)
```

### **Case 2: 주 15시간 이상 정기 근무 바리스타 (1일 6시간, 주 3회)**
```
✅ 원칙: 정규 근로자와 동일하게 처리
✅ 4대 보험: 국민연금, 건강보험, 고용보험, 산재보험 전체 가입
✅ 급여 처리:
   - 근로소득 원천징수 (간이세액표 적용)
   - 연말정산 대상
```

### **Case 3: 1회성 행사 바리스타 (3일 단기 프로젝트)**
```
✅ 원칙: 용역계약 (프리랜서) 가능 여부 검토
✅ 조건:
   - 업무 지시 최소화 (장소·시간만 합의)
   - 바리스타가 자율적으로 업무 수행
   - 1개월 미만 초단기 계약
✅ 급여 처리:
   - 3.3% 원천징수 (사업소득)
   - 사업소득 지급명세서 제출
```

## ⚠️ [리스크 체크리스트: 잘못된 분류 시 불이익]

| 리스크 | 발생 가능 상황 | 패널티 |
|--------|-------------|--------|
| **근로자인데 3.3% 처리** | 주 15시간 이상인데 프리랜서로 분류 | 4대 보험료 소급 납부 + 가산금 |
| **프리랜서인데 4대 보험 가입** | 실제 독립적 업무인데 근로자로 가입 | 보험료 과다 납부 (환급 불가능) |
| **무신고 급여 지급** | 현금 지급 후 지급명세서 미제출 | 가산세 2% + 세무 조사 리스크 |

**⚠️ 핵심 판단 기준: "실제 업무 관계"가 프리랜서인지 근로자인지가 가장 중요합니다.**  
- 출퇴근 시간을 정하고, 업무 방식을 지시하면 → **근로자**
- 결과물만 요구하고, 방법은 자율에 맡기면 → **프리랜서**

## 🚀 [Step-by-Step 실행 가이드]

### **1단계: 바리스타 근로 형태 분류 (이번 주 내)**
```
□ 현재 고용된 바리스타 10명의 주당 근무시간 집계
□ 근무 형태 분류:
   - A그룹: 주 15시간 미만 (고용보험·산재만)
   - B그룹: 주 15시간 이상 (4대 보험 전체)
   - C그룹: 1회성 프로젝트 (프리랜서 가능성 검토)
```

### **2단계: 계약서 정비 (2주 내)**
```
□ 근로계약서 양식 통일 (주 15시간 미만/이상 구분)
□ 프리랜서 용역계약서 별도 작성 (행사성 업무용)
□ 계약서에 근무시간, 업무 내용, 급여 지급일 명시
```

### **3단계: 급여 관리 시스템 구축 (1개월 내)**
```
□ 그룹별 급여 계산 엑셀 템플릿 제작
   - A그룹: 시급 × 근무시간 - 고용·산재보험료
   - B그룹: 시급 × 근무시간 - 4대 보험료 - 소득세
   - C그룹: 용역비 - 3.3%
□ 월별 지급명세서 자동 작성 시스템 (엑셀 or 회계 프로그램)
```

### **4단계: 4대 보험 신고 (즉시)**
```
□ 주 15시간 이상 근무자: 4대 보험 취득신고 (근로 개시일 기준 14일 내)
□ 주 15시간 미만 근무자: 고용·산재보험만 신고
□ 신고 누락 시 소급 적용 + 가산금 발생
```
