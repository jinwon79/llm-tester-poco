**파악된 현황:**

- 리픽싱 조항이 포함된 전환사채(CB) 발행

- 상장 예정 시기

## 💡 핵심 솔루션

### **결론: 원칙적으로 '파생상품부채'로 분류 가능성이 매우 높습니다**

#### 1️⃣ **회계기준 판단 로직 (K-IFRS 1032, 1109 기준)**

| 구분 | 자본 분류 조건 | 리픽싱 조항의 영향 |
|------|---------------|-------------------|
| **고정-고정 원칙** | 고정된 수의 주식을 고정된 금액으로 교환 | ❌ 전환가액이 주가에 연동되어 변동 → 위반 |
| **파생상품 정의** | 기초자산(주가) 변동에 따라 가치 변동 | ✅ 리픽싱 = 주가 연동 조정 → 내재파생상품 해당 |
| **분리 가능성** | 주계약(사채)과 분리하여 측정 가능 | ✅ 전환권을 별도 평가 필요 |

**회계 처리 결과:**
```
발행 시:
차) 현금                    XXX
대) 전환사채(부채)           XXX
대) 파생상품부채(전환권)      XXX

기말 평가:
차) 파생상품평가손실          XXX
대) 파생상품부채             XXX  ← 공정가치 변동분 손익 인식
```

#### 2️⃣ **상장 심사 시 유의사항**

**코스닥 상장심사 관점:**
- 최근 3년간 당기순이익 요건 충족 시 파생상품평가손실이 실적에 악영향
- 상장 직전 사업연도 '자본 잠식' 여부 판단 시 파생부채 증가가 리스크 요인
- 증권신고서 제출 전 반드시 회계법인의 사전 검토 필수

## 🔧 Step-by-Step 실행 가이드

### **단기 조치 (1~2주 내)**

**1단계: 긴급 회계 검토**
- [ ] 기존 계약한 회계법인에 리픽싱 조항 계약서 제출
- [ ] K-IFRS 1032 해석에 따른 '고정-고정 테스트' 수행 요청
- [ ] 유사 상장 사례(테크 기업 CB) 회계 처리 벤치마킹

**2단계: 법률 검토 병행**
- [ ] 상장 주관사 법무법인을 통해 리픽싱 조항의 상장 적격성 검토
- [ ] 금융감독원 회계포탈(K-IFRS) 질의·회신 사례 검색
  - 검색 키워드: "리픽싱", "전환가액 조정", "파생부채"

### **중장기 전략 (상장 前)**

**3단계: 구조 개선 검토**
```
옵션 A: CB 조기 상환 후 재발행
→ 리픽싱 조항 제거한 일반 CB로 전환
→ 회계 처리 단순화 + 상장 심사 리스크 감소

옵션 B: 신주인수권부사채(BW) 전환
→ 신주인수권 자체는 자본으로 분류 가능성
→ 단, 행사가격 조정 조건 재설계 필요

옵션 C: 현 상태 유지 + 공시 강화
→ 사업보고서에 파생부채 평가 방법 상세 공시
→ 주가 안정화 전략으로 리픽싱 발동 최소화
```

**4단계: 상장 IR 자료 보완**
- 파생부채가 일시적 회계 이슈임을 강조
- 기술력·수주 실적 등 본질 가치 어필 자료 강화

## ⚠️ 리스크 체크리스트

### **회계 리스크**
- [ ] **분식회계 오해 가능성**: 파생부채를 자본으로 잘못 분류 시 감사 의견 '한정' 가능
- [ ] **실적 변동성 확대**: 주가 등락에 따라 분기마다 수천만~수억 원의 평가손익 발생
- [ ] **RCPS와의 중복 이슈**: 기존 우선주 발행 시 복합금융상품 회계 처리 복잡도 증가

### **상장 심사 리스크**
| 심사 항목 | 리스크 시나리오 | 대응 방안 |
|----------|----------------|----------|
| 실적 안정성 | 파생손실로 적자 전환 | 영업이익 �흑자 유지 + 별도 설명 자료 |
| 자본 건전성 | 부채비율 급등 | CB→주식 전환 시나리오 시뮬레이션 제출 |
| 공시 충실성 | 복잡한 구조 설명 부족 | 투자설명서에 회계기준 해석 상세 기재 |

### **세무 리스크**
- 파생상품평가손실은 **'손금불산입'** 항목 → 법인세 절감 효과 없음
- 실제 전환 또는 상환 시점에 법인세 영향 재계산 필요
