## 📊 [상황 진단]

### ✅ 현재 귀사 상태
- **제품**: 비침습적 혈당 측정 기술 (광학센서/AI 기반 추정)
- **시장**: 미국 FDA 승인 대기 중 → **특허 침해는 FDA 승인과 별개로 민사소송 리스크 발생**
- **경쟁 환경**: Abbott(FreeStyle Libre), Dexcom 등 글로벌 기업의 **특허 포트폴리오 밀집 지역**

### ⚠️ 핵심 리스크
1. **미국은 '선발명주의→선출원주의'로 전환**되었으나, 과거 특허(2013년 이전)는 선발명주의 적용 가능
2. FDA 승인 후 제품 출시 시점에 경쟁사의 **특허소송(Patent Infringement Lawsuit)** 가능성
3. 특허 침해 판결 시 **손해배상 + 판매금지(Injunction)** 이중 타격

## 🎯 [FTO(Freedom To Operate) 조사 Step-by-Step]

### **Phase 1: 사전 분석 (2-3주)**

| 단계 | 실행 내용 | 담당 |
|------|-----------|------|
| **1-1** | 자사 기술의 핵심 구성요소 정리<br>- 센서 방식(광학/전기화학)<br>- 알고리즘(AI 모델 구조)<br>- 하드웨어 설계 | 내부 R&D팀 |
| **1-2** | 미국 특허청(USPTO) DB 검색<br>- 키워드: "non-invasive glucose monitoring", "wearable biosensor"<br>- 분류코드: A61B 5/145 (혈당 측정) | 특허 검색 전문업체<br>(예: PatSnap, Innography) |
| **1-3** | 경쟁사 특허 포트폴리오 맵핑<br>- Abbott, Dexcom, Medtronic 보유 특허 우선 분석 | 특허법무법인 |

### **Phase 2: 법률 검토 (4-6주)**

```
[필수 확인 사항]
✓ 특허 청구항(Claim) 분석
  → 독립항(Independent Claim)과의 유사성 80% 이상 시 침해 위험
✓ 특허 유효기간 (출원일 기준 20년)
✓ 특허권자의 라이선싱 정책
✓ 과거 소송 이력 (PACER 시스템 검색)
```

**💡 추천 협력 파트너**
- **미국 현지 IP Law Firm**: Fish & Richardson, Morrison & Foerster (의료기기 전문)
- **한국**: 특허법인 다래/코리아나 (해외 FTO 경험 보유)
- **비용**: 약 $15,000~$30,000 (기술 복잡도에 따라 변동)

## 🛡️ [유사 특허 발견 시 대응 전략]

### **전략 A: 회피 설계(Design-Around)**

#### 📐 **설계 변경 원칙**
| 침해 유형 | 회피 설계 방법 | 예시 (귀사 적용) |
|-----------|---------------|-----------------|
| **센서 구조** | 측정 파장 대역 변경 | 근적외선(NIR) → 중적외선(MIR)으로 전환 |
| **알고리즘** | 학습 데이터셋 구성 변경 | 개인화 보정 알고리즘 추가 (경쟁사 특허 회피) |
| **착용 방식** | 접촉 부위 변경 | 손목 → 상완(upper arm) 또는 귀(earlobe) |

#### ✅ **법적 안전성 확보 절차**
```
1단계: 변경된 설계로 신규 FTO 조사 재실시
2단계: 특허법무법인의 '비침해 의견서(Non-Infringement Opinion)' 발급
        → 선의의 침해(Willful Infringement) 방어 근거
3단계: 자사 기술의 특허 출원 (미국 가출원 우선 추천)
        → 방어적 특허 포트폴리오 구축
```

### **전략 B: 라이선싱 협상**

만약 회피 설계가 기술적으로 불가능하거나 비용 효율이 낮을 경우:

- **선제적 협상**: 제품 출시 전 특허권자에게 라이선스 요청
- **Cross-License 제안**: 귀사의 AI 알고리즘 특허와 교환 조건 제시
- **협상 레버리지**: "한국/아시아 시장 독점 공급 계약" 등 부가가치 제공

## ⚖️ [리스크 체크리스트]

### 🚨 **즉시 점검 필요 사항**
- [ ] 현재 제품이 이미 한국에서 판매 중인가? → **공개된 기술은 특허 무효화 근거 가능**
- [ ] 자사 특허 출원 여부? → 미출원 시 **즉시 미국 가출원(Provisional Application) 진행**
- [ ] 투자 계약서에 "특허 침해 면책 조항" 포함 여부? → 없다면 VC와 재협상 필요

### ⏱️ **타임라인 주의**
```
❌ 위험: FDA 승인 → 제품 출시 → 특허 소송 → 손해배상
✅ 안전: FTO 조사 → 회피 설계 → 특허 출원 → FDA 승인 → 제품 출시
```
**최소 6개월의 선행 기간** 확보를 권장합니다.

## 🚀 [Step-by-Step 실행 가이드]

### **[즉시 실행: 1-2주 내]**
1. **특허 전문 법무법인 3곳 컨택** (견적 및 경험 비교)
   - 질문 리스트: "의료기기 FTO 경험", "Dexcom/Abbott 대응 사례", "비용 구조"
2. **자사 기술 문서화**
   - R&D팀이 센서 원리, 알고리즘, 하드웨어 설계를 '특허 청구항 형식'으로 정리
3. **예산 확보**
   - FTO 조사: $20,000
   - 회피 설계 R&D: 2,000만 원
   - 미국 특허 출원: $10,000

### **[단기: 2-3개월]**
4. **FTO 보고서 수령 및 리스크 분석**
   - 침해 가능성 60% 이상 특허 → 회피 설계 착수
   - 침해 가능성 30% 이하 → 비침해 의견서 발급으로 진행
5. **변경된 설계로 시제품 재제작**
   - 성능 검증 후 FDA 승인 서류에 반영

### **[중장기: 6개월~1년]**
6. **방어적 특허 포트폴리오 구축**
   - 회피 설계 기술로 PCT 국제 출원
   - 미국/유럽/중국 동시 진입 대비
7. **경쟁사 모니터링 시스템 구축**
   - 분기별 신규 특허 출원 추적 (PatSnap 구독 추천)
