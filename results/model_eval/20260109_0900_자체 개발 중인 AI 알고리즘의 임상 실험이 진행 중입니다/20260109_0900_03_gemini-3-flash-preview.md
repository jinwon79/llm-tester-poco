# gemini-3-flash-preview  응답 (3/3회차)

## 응답 내용
**(주)바이오에이아이의 지속 가능한 성장을 위해, 회계/재무 전문가(CPA)를 주축으로 하되 IT/기술(CTO) 관점을 결합하여 답변드립니다.**

현재 (주)바이오에이아이는 시리즈 B 투자를 앞둔 중요한 시점에 있습니다. 투자자들과 회계 감사인은 무형자산(개발비) 비중이 급격히 늘어난 점을 '재무 건전성 리스크'로 볼 가능성이 높습니다. 특히 바이오·의료 AI 분야는 기술적 실현 가능성의 판단 기준이 매우 엄격하므로, 단순한 주장보다는 **'객관적이고 외부에서 검증된 증빙'**을 확보하는 것이 핵심입니다.

---

### [상황 진단]
*   **핵심 쟁점:** 개발비 자산화의 6가지 요건 중 '기술적 실현 가능성(Technical Feasibility)' 입증.
*   **리스크 요인:** 임상 실험 결과가 나오기 전 단계에서 인건비와 서버 비용을 자산화했을 경우, 감사인이 이를 '연구비(비용)'로 처리하라고 권고할 수 있으며, 이는 당기순이익의 급격한 감소로 이어질 수 있습니다.
*   **추가 필요 정보:** 현재 진행 중인 임상 단계(식약처 승인 여부 등)와 사내 '프로젝트별 타임시트(Time-sheet)' 관리 여부를 확인해야 합니다.

---

### [핵심 솔루션: 기술적 실현 가능성 입증 증빙 리스트]

회계 감사 시 기술적 실현 가능성을 입증하기 위해 반드시 준비해야 할 핵심 증빙 자료는 다음과 같습니다.

| 구분 | 핵심 증빙 항목 | 상세 내용 및 용도 |
| :--- | :--- | :--- |
| **외부 검증** | **임상시험계획(IND) 승인서** | 식약처 등 규제기관의 승인을 받았다는 사실은 기술적 실현의 초기 단계를 입증하는 가장 강력한 증거입니다. |
| **외부 검증** | **중간 임상 결과 보고서** | 현재 진행 중인 임상의 유효성 지표(AUC, 정확도, 재현율 등)가 목표치에 부합함을 보여주는 데이터입니다. |
| **내부 기술** | **상세 기술 설계서 (SRS/TDD)** | AI 알고리즘의 아키텍처, 데이터 전처리 과정, 학습 모델의 구조가 명시된 문서입니다. |
| **내부 기술** | **기술 로드맵 및 완료 확약서** | 개발 완료 예정일, 필요한 추가 자원, 최종 상용화 계획이 포함된 경영진 승인 문서입니다. |
| **지식재산권** | **특허 출원/등록 증명서** | 해당 알고리즘의 독창성과 기술적 우위를 법적으로 보호받고 있음을 입증합니다. |
| **비용 연관성** | **프로젝트별 인건비 배분표** | 15명의 연구원이 해당 프로젝트에 투입된 시간(Time-sheet)과 급여를 매칭한 자료입니다. |
| **비용 연관성** | **서버 자원 사용 내역서** | 클라우드(AWS/GCP 등) 사용량 중 해당 AI 모델 학습에 사용된 인스턴스 비용을 구분한 명세입니다. |

### [Step-by-Step 실행 가이드]

#### 1단계: 개발비 자산화 시점(Milestone) 공식화 (즉시 시행)
*   내부 회계 규정에 '연구 단계'와 '개발 단계'를 명확히 구분하는 기준을 수립하십시오.
*   의료 AI의 경우, 통상적으로 **'식약처 임상시험계획(IND) 승인'** 또는 **'확정 임상시험 단계 진입'**을 자산화 시작 시점으로 설정하는 것이 감사 대응에 유리합니다.

#### 2단계: 서버 비용의 '자본적 지출' 증빙화 (단기)
*   서버 비용(GPU 사용료 등)은 단순 운영비와 구분되어야 합니다.
*   클라우드 관리 콘솔에서 프로젝트별 태그(Tagging)를 설정하여, '서비스 운영' 비용이 아닌 '모델 개발/학습'에 투입된 비용임을 소명할 수 있는 로그 자료를 추출해 두십시오.

#### 3단계: 기술 감정 또는 외부 전문가 의견서 확보 (시리즈 B 직전)
*   감사인이 기술력을 판단하기 어려워할 경우, 기술신용평가(TCB) 결과나 해당 분야 권위자(대학병원 교수 등)의 기술 실현 가능성 검토 의견서를 첨부하면 공신력이 대폭 상승합니다.

### [리스크 체크리스트]

1.  **손상 검사(Impairment Test):** 임상 실험이 실패하거나 지연될 경우, 계상된 무형자산을 한꺼번에 비용(손실) 처리해야 하는 '빅 배스(Big Bath)' 리스크가 있습니다.
2.  **보수적 회계 처리:** 시리즈 B 투자자가 '부풀려진 자산'보다 '클린한 재무제표'를 선호할 수 있습니다. 자산화 비중이 업계 평균(바이오/소프트웨어)보다 과도하게 높지 않은지 점검하십시오.
3.  **인건비 중복 계상:** 연구원이 정부지원사업(R&D) 과제를 수행 중이라면, 정부지원금으로 지급된 인건비는 자산화 대상에서 제외해야 합니다.
