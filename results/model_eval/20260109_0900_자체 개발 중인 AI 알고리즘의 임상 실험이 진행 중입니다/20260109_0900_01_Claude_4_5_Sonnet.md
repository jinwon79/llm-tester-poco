## [Step-by-Step 실행 가이드]

### **PHASE 1: 즉시 실행 (투자 실사 전 2주)**

**1주차: 증빙 Gap Analysis**
```
□ 현재 보유 증빙 목록 작성
□ 위 5대 카테고리별 누락 문서 식별
□ 긴급도 순위 설정 (IRB 승인 > 기술평가 > 사업화 계획)
```

**2주차: 긴급 보완 조치**
```
□ 기술 로드맵 문서화 (CTO 주관, PPT + 수치 데이터)
□ 임상시험 진행률 정량화 (예: 피험자 모집 80% 완료)
□ 외부 자문의 서면 의견서 확보 (최소 3명)
□ 회계법인과 사전 협의 (자산화 적격성 검토)
```

### **PHASE 2: 중기 대응 (3개월 이내)**

**기술적 실현 가능성 강화**
- **알고리즘 성능 벤치마킹**: 기존 FDA 승인 AI 의료기기와 비교 분석 보고서 작성
- **데이터셋 품질 증명**: 학습 데이터의 출처(대학병원 협약), 전처리 방법론, 라벨링 정확도 문서화
- **재현 가능성 확보**: 독립된 테스트셋 검증 결과 공식 기록

**규제 대응 가속화**
- **식약처 사전 상담 신청**: 품목 분류, 임상시험 계획 피드백 확보
- **ISO 13485 준비**: 의료기기 품질관리시스템 구축 착수
- **개인정보보호**: 임상데이터 비식별화 절차 감사 대응 준비

### **PHASE 3: 장기 전략 (시리즈 B 이후)**

**지속적 증빙 체계 구축**
```
1. 분기별 기술 마일스톤 달성률 리포트 (이사회 보고용)
2. 임상시험 단계별 회계처리 정책 명문화
3. 개발비 자산화 중단 시점 사전 설정 (예: 3등급 인증 실패 시)
```
