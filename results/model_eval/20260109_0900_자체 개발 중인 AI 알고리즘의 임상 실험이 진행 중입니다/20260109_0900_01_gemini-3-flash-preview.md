# gemini-3-flash-preview  응답 (1/3회차)

## 응답 내용
(주)바이오에이아이의 성공적인 시리즈 B 투자 유치와 투명한 재무제표 관리를 위해, **회계/재무(CPA) 전문가를 주축으로 하되 IT/기술 전문가의 관점을 결합하여** 답변드립니다.

### [전문가 선언]
바이오/AI 헬스케어 분야의 회계 감사는 '개발비(무형자산) 인정' 여부가 기업 가치 평가의 핵심입니다. K-IFRS(또는 K-GAAP) 상 연구단계와 개발단계를 엄격히 구분해야 하며, 특히 **'기술적 실현 가능성'**은 단순히 "개발 중이다"라는 주장만으로는 부족하고, 객관적인 제3자가 검증 가능한 문서화된 증거가 필수적입니다.

---

### [상황 진단]
- **현재 상황:** 시리즈 B를 앞두고 자산 비중이 높아진 상태입니다. 이는 자본 잠식을 방어하고 기업 가치를 높이는 효과가 있으나, 감사 시 '비용의 자산화'에 대한 엄격한 잣대가 적용될 시점입니다.
- **필요 정보:** 현재 진행 중인 임상 단계(1, 2, 3상 또는 확증 임상 여부)와 식약처(MFDS) 등 규제 기관의 인허가 진행 단계에 대한 구체적 데이터가 추가로 필요합니다.

### [핵심 솔루션: 기술적 실현 가능성 입증 증빙]

회계사가 감사 시 집중적으로 검토하는 **5가지 핵심 증빙 목록**입니다.

| 구분 | 핵심 증빙 항목 | 상세 내용 |
| :--- | :--- | :--- |
| **1. 기술적 증빙** | **SRS 및 아키텍처 설계서** | AI 모델의 구조, 입력 데이터의 정의, 목표 성능 수치(정확도, 민감도 등)가 명시된 문서 |
| | **버전 관리 기록 (Git 등)** | 개발자의 투입 시점과 코드 커밋 이력을 통해 실제 개발 행위가 지속되었음을 증명 |
| **2. 임상/규제 증빙** | **임상시험계획서(Protocal) 및 승인서** | IRB(임상시험심사위원회) 승인서 및 식약처 임상시험계획 승인서 (가장 강력한 증빙) |
| | **중간 임상 결과 보고서** | 현재 진행 중인 임상에서 유의미한 수치가 도출되고 있음을 보여주는 내부/외부 리포트 |
| **3. 경제적 증빙** | **시장 분석 및 수익 모델 보고서** | 기술 완성 후 대학병원/검진센터와의 계약 의향서(LOI) 또는 예상 매출 추정 근거 |
| **4. 인건비 증빙** | **타임시트(Time-Sheet)** | 15명의 연구원이 해당 프로젝트에 투입된 시간 비중을 일별/주별로 기록한 서류 |
| **5. 서버비 증빙** | **리소스 사용 내역서** | 클라우드(AWS/GCP 등) 청구서 중 해당 프로젝트 인스턴스/스토리지에 할당된 비용 상세 |

### [Step-by-Step 실행 가이드]

#### 1단계: 개발단계 확정 및 문서 소급 정비 (단기)
*   **Action:** 연구단계(비용 처리)와 개발단계(자산 처리)의 구분 시점을 명확히 정의하십시오. AI 의료기기의 경우 보통 **'확증 임상시험 승인'** 시점을 개발단계의 시작으로 보는 경우가 많습니다. 그 이전의 비용은 과감히 비용 처리하여 리스크를 줄여야 합니다.

#### 2단계: 외부 전문가 감정 및 기술평가 (중기)
*   **Action:** 시리즈 B 투자 전, 전문 기술평가 기관으로부터 **'기술성 평가'**를 미리 받아두십시오. 이는 감사인에게 기술적 실현 가능성을 설득할 수 있는 강력한 보조 자료가 됩니다.

#### 3단계: 임상 데이터와 회계 장부의 동기화 (상시)
*   **Action:** 임상 실패 리스크에 대비하여, 만약 임상 결과가 부정적으로 나올 경우 즉시 해당 자산을 손상차손(Impairment) 처리할 수 있는 내부 프로세스를 구축하십시오.
