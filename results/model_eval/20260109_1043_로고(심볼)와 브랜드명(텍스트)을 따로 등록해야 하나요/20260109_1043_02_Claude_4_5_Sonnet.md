## 응답 내용
# [법률·IP 전문가] 상표권 전략 컨설팅

**"브루잉 카페"의 프랜차이즈 확장을 위한 상표권 등록 전략을 설계해드리겠습니다.**

---

## 📋 [상황 진단]

**파악된 기업 현황:**
- 스페셜티 커피 B2C 브랜드 (구독+오프라인)
- 가맹사업 진출 계획 → **상표권 확보가 필수 단계**
- 로고(심볼)+브랜드명(텍스트) 디자인 완성 상태

**핵심 질문 분석:**
로고와 브랜드명의 **분리 등록 vs 결합 등록** 전략 결정이 필요한 상황입니다.

**추가 확인 필요 정보:**
더 정확한 전략 수립을 위해 다음을 알려주시면 맞춤형 조언이 가능합니다:
- 로고 디자인 내 텍스트 비중 (심볼 중심 vs 텍스트 중심)
- 향후 3년 내 해외 진출 계획 여부
- 예산 범위 (상표 출원 건당 약 10~15만원, 변리사 수수료 별도)

---

## 💡 [핵심 솔루션]

### ✅ **결론: "3건 분리 등록"을 강력히 권장합니다**

| 구분 | 등록 대상 | 보호 범위 | 권장 이유 |
|------|-----------|-----------|-----------|
| **1안 (필수)** | 브랜드명 **"브루잉 카페"** (문자만) | 가장 넓은 보호 | 타인의 유사 명칭 사용 차단 |
| **2안 (필수)** | 로고 심볼 (이미지만) | 시각적 고유성 보호 | 리브랜딩 시에도 심볼 독점권 유지 |
| **3안 (권장)** | 로고+브랜드명 결합형 | 실제 사용 형태 보호 | 패키지·간판에 쓰이는 완성 형태 |

## 🎯 [전략별 비교 분석]

### 📌 **분리 등록의 장점 (추천)**

**1. 유연한 브랜드 활용**
```
예시 시나리오:
- 구독 서비스: 심볼만 사용 (앱 아이콘, 패키지)
- 프랜차이즈 간판: 텍스트 중심 사용
- 협업 상품: 심볼+타 브랜드명 조합
```
→ 각 상황에서 **별도 권리 주장 가능**

**2. 리스크 분산**
- 만약 "브루잉 카페" 명칭이 선등록 상표와 유사해 거절되더라도
- **로고 심볼은 별도로 등록 가능** (디자인이 고유하면)

**3. 법적 방어력 강화**
```
침해 사례 발생 시:
❌ 결합형만 등록: "우리는 심볼만 따라했고 글자는 달라요" → 방어 곤란
✅ 분리 등록: 심볼 침해/명칭 침해 각각 소송 가능
```

---

### ⚠️ **결합형 단독 등록의 위험성**

| 상황 | 문제점 |
|------|--------|
| 타 업체가 "브루잉카페" 텍스트만 사용 | 결합형 권리로는 제재 어려움 |
| 로고 리뉴얼 필요 시 | 새로운 출원 필요 (기존 권리 활용 불가) |
| 가맹점 간판 다양화 | 결합형과 다른 형태 사용 시 보호 공백 |

## 📝 [Step-by-Step 실행 가이드]

### 🔹 **STEP 1: 출원 전 사전 조사 (D+0 ~ D+7)**

**필수 작업:**
1. **선등록 상표 검색**
   - 특허청 키프리스(KIPRIS) 무료 검색: http://www.kipris.or.kr
   - 검색어: "브루잉", "브루잉카페", "Brewing Cafe"
   - 유사 업종(커피, 음료, 가맹업) 기준으로 필터링

2. **유사 상표 판단 기준**
   ```
   ⚠️ 주의 대상:
   - 브루잉 ○○○ (예: 브루잉 스튜디오)
   - ○○○ 브루잉 카페
   - Brewing + 다른 단어 조합
   ```

**자체 검색이 어려울 경우:**
- 변리사 사무소 무료 상담 활용 (출원 전제 시 검색 무료 제공)

---

### 🔹 **STEP 2: 출원 범위 및 상품류 결정 (D+7 ~ D+10)**

**등록 대상 3건:**

| 출원 건 | 상표 형태 | 비용 | 우선순위 |
|---------|-----------|------|----------|
| 1건 | "브루잉 카페" (문자) | 약 12만원 | ★★★ 최우선 |
| 2건 | 로고 심볼 (이미지) | 약 12만원 | ★★★ 최우선 |
| 3건 | 로고+텍스트 결합 | 약 12만원 | ★★☆ 권장 |

**상품류 선택 (니스분류 기준):**
```
✅ 필수 등록 상품류:
- 제30류: 커피, 커피 음료, 원두
- 제43류: 커페테리아, 카페, 음식점업
- 제35류: 가맹점 모집 및 운영 (프랜차이즈)

🔹 선택 상품류 (예산 허용 시):
- 제21류: 커피 추출기, 머그컵 (굿즈 판매 대비)
- 제9류: 모바일 앱 (구독 서비스 관련)
```

**예산 최적화 팁:**
- 1개 출원에 여러 상품류 포함 시 추가 비용 발생 (류당 +3~5만원)
- **초기에는 30류+43류+35류만 우선 확보** 권장

---

### 🔹 **STEP 3: 출원 서류 준비 및 접수 (D+10 ~ D+14)**

**자가 출원 vs 변리사 의뢰:**

| 방식 | 비용 | 장점 | 단점 |
|------|------|------|------|
| 자가 출원 | 10~12만원/건 | 비용 절감 | 거절 시 대응 어려움 |
| 변리사 의뢰 | 30~50만원/건 | 전문적 대응, 성공률↑ | 초기 비용 부담 |

**추천 방식:**
```
💡 절충안: 
1. 1~2건은 변리사 의뢰 (문자상표 + 중요 로고)
2. 3건(결합형)은 자가 출원 시도
   → 거절 시 변리사 보정 서비스만 구매 (5~10만원)
```

**출원 서류:**
- 상표등록출원서 (특허청 양식 다운로드)
- 상표 견본 (5cm x 5cm, 300dpi 이상 JPG/PNG)
- 사업자등록증 사본
- 대리인 선임장 (변리사 의뢰 시)

**온라인 출원 경로:**
특허로 (www.patent.go.kr) → 전자출원 → 상표 출원

---

### 🔹 **STEP 4: 심사 대응 및 등록 (D+180 ~ D+365)**

**심사 타임라인:**
```
출원 후 6~12개월: 심사관 심사
→ 거절이유 통지 가능 (약 30% 확률)
→ 의견서 제출 (30일 이내)
→ 등록 결정 → 등록료 납부 (3년분 약 21만원)
```

**거절 대응 예시:**
```
거절 사유: "브루잉"이 커피 제조 방법을 나타내 식별력 없음

💡 대응 전략:
- 사용 증거 제출 (매출, SNS, 언론보도 등)
- "카페"와의 결합으로 고유성 주장
- 디자인 요소 강조 (로고 심볼의 독창성)
```
