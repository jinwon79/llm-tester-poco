## 🎯 **핵심 솔루션: 3단 방어 전략 추천**

| 구분 | 등록 방식 | 보호 범위 | 추천도 |
|------|----------|----------|--------|
| **① 결합상표** | 로고+텍스트 일체형 | 정확히 이 조합만 보호 | ★★★★★ 필수 |
| **② 문자상표** | "브루잉 카페" 텍스트만 | 폰트 무관, 텍스트 독점 | ★★★★★ 필수 |
| **③ 도형상표** | 심볼(로고 이미지)만 | 시각적 요소 독점 | ★★★☆☆ 선택 |

### **결론: ①+② 동시 등록이 최적 (③은 예산 여유시 추가)**

## 📋 **Step-by-Step 실행 가이드**

### **[STEP 1] 즉시 착수 (D+0 ~ D+7)**
```
✅ 1순위: 결합상표(로고+텍스트) 출원
   → 현재 확정된 비주얼 그대로 등록
   → 상품류 구분: 제30류(커피), 제43류(카페), 제35류(가맹)

✅ 2순위: 문자상표 "브루잉 카페" 출원
   → 폰트·색상 무관하게 이름 자체를 독점
   → 동일 상품류 3개 구분 출원
```

**💡 왜 둘 다 해야 하나요?**
- **결합상표만 있으면**: 타사가 비슷한 로고에 "브루잉 카페"를 써도 막기 어려움
- **문자상표만 있으면**: 로고를 똑같이 베껴도 글자만 바꾸면 침해 입증 곤란
- **두 개 병행**: 이름도, 디자인도 모두 방어 → **가맹점 보호 + 모방 브랜드 차단**

---

### **[STEP 2] 선행 상표 조사 (D+1 ~ D+3)**
출원 전 반드시 확인해야 할 사항:

| 체크 항목 | 확인 방법 | 리스크 |
|----------|----------|--------|
| "브루잉" 포함 동일/유사 상표 | 특허청 키프리스(kipris.or.kr) | 거절 확정 → 비용 손실 |
| 커피/카페 업종 등록 현황 | 상품류 30, 43, 35류 검색 | 분쟁 시 수억원 손해배상 |
| 영문 "Brewing Cafe" 등록 여부 | 한글·영문 교차 검색 필수 | 해외 진출 시 역공 가능성 |

**🔍 직접 검색 어려우시면**:  
→ 변리사 사전조사 비용 **20~30만원** (출원 실패 방지 투자)

---

### **[STEP 3] 출원 및 모니터링 (D+7 ~ 12개월)**

#### **비용 구조**
```
📌 자체 출원 (특허청 직접)
- 결합상표: 6.2만원/1류당
- 문자상표: 6.2만원/1류당
- 3개 류 출원 시 약 37만원

📌 변리사 대행 (권장)
- 수수료 포함 약 120~150만원/2건
- 거절 시 의견서 대응 포함
```

#### **타임라인**
- 출원 후 **9~12개월** 심사  
- 이의신청 기간 **2개월**  
- 등록 후 **10년** 유효 (갱신 가능)
