## 응답 내용
# 📋 법률·투자계약 전문가로서 답변드리겠습니다

**[상황 진단]**
시리즈 A 단계에서 30억 규모 투자 협상 중, VC의 이사 지명권 조항에 대한 대응 전략이 필요한 상황입니다. 하드웨어 중심 기업 특성상 기술 보호와 경영 자율성 확보가 특히 중요합니다.

---

## 🎯 핵심 솔루션

### 1️⃣ 이사 지명권의 법적 성격

| 구분 | 내용 |
|------|------|
| **법적 근거** | 상법 제382조 (주주총회의 이사 선임권) |
| **계약 효력** | VC와의 투자계약서 = 사법(私法)상 계약이므로 **합의한 조건은 이행 의무 발생** |
| **거부 시 리스크** | • 계약 위반으로 투자금 전액 회수 조항 발동 가능<br>• 채무불이행 책임(손해배상 청구)<br>• 후속 투자 유치 시 신뢰도 하락 |
| **현실적 평가** | **"서명 후 일방적 거부"는 사실상 불가능** |

⚠️ **중요**: 계약서에 명시된 조항은 법적 구속력을 가지므로, **서명 전 협상 단계**에서 조건을 조정하는 것이 유일한 해법입니다.

## 📌 Step-by-Step 실행 가이드

### ✅ **STEP 1: 계약 전 협상 (現 단계 - 최우선 조치)**

#### A. 이사 지명권 조건 완화 협상 전략

**1) 대안 조건 제시 (순서대로 협상)**
```
제1안: 사외이사 → 옵저버(Observer) 전환
       - 이사회 참석·발언권 O, 의결권 X
       - 기업 입장: 경영권 침해 최소화
       - VC 입장: 모니터링 목적 달성
       
제2안: 조건부 지명권 (Trigger 조항)
       - 발동 조건: 연간 매출 목표 미달 2회 연속 시에만
       - 정상 성장 시 지명권 자동 소멸
       
제3안: 이사 자격 요건 명문화
       - 모빌리티/하드웨어 업계 10년 이상 경력자로 한정
       - 경쟁사 임직원 배제 조항 추가
       - 후보 3인 중 창업자가 최종 선택권 보유
```

#### B. 협상 카드 활용법

| 당신의 협상 카드 | 활용 전략 |
|------------------|-----------|
| **시리즈 A 단계의 성장성** | "현재 공유 킥보드 업체 2곳과 파일럿 계약 완료, 향후 12개월 내 흑자 전환 가능한 BM" 강조 |
| **타 투자자 관심** | (실제 있다면) "다른 VC에서도 제안이 있으나, 귀사의 네트워크를 우선 고려 중" |
| **기술 경쟁력** | "무선 충전 특허 2건 출원 중, 기술 유출 방지 위한 최소한의 보호 필요" |

### ✅ **STEP 2: 불가피하게 수용 시 - 보호장치 설치**

#### 필수 삽입 조항 (법무법인 검토 필수)

**1) 의결권 제한 조항 (Reserved Matters)**
```
[예시 문구]
"VC 지명 이사는 다음 사항에 대해서만 의결권을 행사한다:
 ① 연간 사업계획 및 예산 승인
 ② 50억 이상 차입/투자 건
 ③ 정관 변경, 합병·분할 등 조직 재편
 
 * 일상적 경영 사항(인사, 제품 개발 방향 등)은 
   창업자 중심 경영진이 단독 결정"
```

**2) Drag-Along 방어 조항**
```
"VC 지명 이사는 회사 매각(Exit) 시 
 창업자 동의 없이 매각 강제 의결 불가"
```

**3) 이익충돌 방지**
```
"경쟁사 투자 포트폴리오 보유 시 
 해당 이사는 기술정보 접근 제한"
```

| **Exit 전** | • 우호 지분 30% 이상 확보 (엔젤, 전략적 투자자)<br>• 이사회 의석 과반 유지 전략 |

### 🔴 절대 피해야 할 상황
- [ ] **"일단 계약하고 나중에 거부"** → 즉시 소송 리스크
- [ ] **투자계약서 단독 검토** → 업계 관행 모르면 불리한 조항 간과
- [ ] **구두 합의만 믿기** → 반드시 계약서 명문화

```

□ VC의 투자 포트폴리오에 경쟁사가 있는가?
  (있다면 → 정보 차단 조항 필수)
  
□ 이사회 의결 정족수는? (과반? 3분의 2?)
  (높을수록 → VC 영향력 증가)
```
