## 🎯 핵심 솔루션: 5단계 보호 장치

### **1단계: 정관에 명시할 필수 의사결정 구조**

```
┌─────────────────────────────────────┐
│  권한 분산형 거버넌스 설계          │
└─────────────────────────────────────┘

[A안] 최종 결정권자 지정 방식
CEO에게 Casting Vote(결정권) 부여
→ 정관 예시: "주주총회 의결이 동수일 경우 
   대표이사가 최종 결정권을 갖는다"

[B안] 의사결정 영역별 분리 통치
- 전략/재무: CEO 주도
- 기술/제품: CTO 주도  
- 영업/마케팅: COO 주도
→ 각자의 전문 영역에서는 단독 결정 가능
```

**추천: A안 + B안 하이브리드**  
일상 경영은 B안으로 분권화하되, 핵심 사안(자금 조달, M&A, 청산 등)은 A안의 Casting Vote 적용

---

### **2단계: 베스팅(Vesting) 조항 - 지분 확정 조건**

| 구분 | 내용 | 법적 효과 |
|------|------|----------|
| **Cliff Period** | 1년간 의무 재직 | 1년 내 퇴사 시 지분 0% 환수 |
| **Vesting 기간** | 4년 균등 배분 | 매년 25%씩 권리 확정 |
| **가속 조건** | IPO/M&A 시 100% 즉시 확정 | Exit 시 창업자 보호 |
| **Bad Leaver** | 횡령/배임/경업 시 시가 50% 강제 매도 | 리스크 사전 차단 |

**정관 예시문:**
```
제○조 (주식의 양도제한)
① 본 회사의 창업 주주는 회사 설립일로부터 1년의 Cliff 기간과 
   이후 3년간의 Vesting 기간을 거쳐 지분을 확정한다.
② Cliff 기간 중 퇴사 시 보유 주식 전량을 액면가로 회사에 환매한다.
```

---

### **3단계: 교착 상태(Deadlock) 해소 메커니즘**

#### 🔹 **Russian Roulette 조항**
```
3개월 이상 의사결정 불가 시:
1. 먼저 제안한 1인이 타 2인에게 가격 제시
2. 타 2인은 "그 가격에 사거나 or 팔거나" 선택
3. 30일 내 결정
```

#### 🔹 **중재 조항** (소송 방지)
```
정관에 명시:
"주주 간 분쟁 발생 시 대한상사중재원의 
 중재 절차를 거치며, 중재 판정은 최종적이다"
```

---

### **4단계: 선매권(Right of First Refusal) & 동반매도권(Tag-Along)**

| 조항명 | 내용 | 목적 |
|--------|------|------|
| **선매권** | 지분 매각 시 타 주주에게 우선 매수 기회 | 외부인 유입 차단 |
| **동반매도권** | 대주주 지분 매각 시 소수 주주도 동일 조건 참여 가능 | 소수 주주 보호 |
| **동반매수권** | 소수 주주 매각 시 대주주가 동일 조건 전량 매수 | 지분 분산 방지 |

**정관 삽입 예시:**
```
제○조 (주식양도 시 선매권)
주주가 제3자에게 주식을 양도하고자 할 때에는 
타 주주에게 서면으로 통지하고, 타 주주는 
통지일로부터 30일 이내에 동일 조건으로 
우선 매수할 권리를 갖는다.
```

### **5단계: 경업금지 및 핵심인력 보호**

```yaml
정관 + 별도 주주간계약서(SHA) 명시:

경업금지:
  - 재직 중 + 퇴사 후 2년간
  - 동종 업계 창업/임원 취임 금지
  - 위반 시 손해배상 + 지분 강제 매각

기밀유지:
  - 고객 데이터, 알고리즘, 사업 계획 영구 보호
  - 위반 시 형사 고소 + 민사 손배

핵심인력 유출 방지:
  - 퇴사 시 개발팀 5인 이상 스카우트 금지
```

## 🚀 Step-by-Step 실행 가이드

### ✅ **즉시 실행 (이번 주)**
1. **3인 긴급 회의 소집**
   - 안건: "투자 유치 전 정관 보완 필수 사항"
   - 결과물: 위 5가지 조항 중 수용 가능 범위 합의

2. **법무법인 자문 예약**
   - 비용: 100~300만원 (스타트업 패키지)
   - 준비물: 현 정관, 주주명부, 투자 Term Sheet
   - 추천: 법무법인 **광장, 율촌, 세종**(스타트업 팀 보유)

### 📅 **1개월 내 완료**
3. **정관 변경 주주총회**
   - 특별결의(2/3 이상) 필요
   - 법원 등기 완료 (변경일로부터 2주 내)

4. **주주간계약서(SHA) 체결**
   - 정관에 넣기 어려운 세부 조항 포함
   - 3인 전원 날인 + 공증(추천)

### 🎯 **시드 투자 유치 시 협상 포인트**
5. **투자자 요구 조건 대응**
   ```
   흔한 VC 요구사항:
   - 이사회 의석 1석 요구 → 3인+투자자 1인 = 4인 체제
   - 거부권(Veto Right) 조항 → 핵심 사안 5가지만 합의
   - 우선주 전환 조건 → 표준 Term Sheet 참조
   ```
