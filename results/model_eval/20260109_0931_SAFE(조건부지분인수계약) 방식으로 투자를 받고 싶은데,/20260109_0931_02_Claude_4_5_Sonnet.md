## 응답 내용
# 법률 분야 전문가 (기업법무/투자계약 전문)로서 답변드리겠습니다

---

## 📋 [상황 진단]

**뉴런 헬스**는 전형적인 Pre-Series A 단계의 헬스테크 스타트업으로, SAFE(Simple Agreement for Future Equity) 방식의 투자 제안을 검토 중이신 상황입니다.

### 추가 확인이 필요한 정보:
- 현재 법인 설립 후 경과 기간 (3년 이내 여부)
- 기존 투자 유치 이력 및 현재 주주 구조
- 제안받은 SAFE의 구체적 조건 (밸류에이션 캡, 할인율 등)
- 향후 6~12개월 내 Series A 투자 유치 계획 유무

---

## 🎯 [핵심 솔루션]

### ✅ **결론부터 말씀드리면: 한국에서 SAFE는 '사용 가능하나, 완벽한 법적 검증은 필요'합니다.**

| 구분 | 내용 |
|------|------|
| **법적 지위** | 명시적 법령 근거 없음 (미국 Y Combinator 고안 계약) |
| **실무 활용** | 국내 스타트업 생태계에서 활발히 사용 중 |
| **주요 리스크** | ① 전환 시점의 분쟁 가능성 ② 세무 해석의 불확실성 |
| **대안** | 전환사채(CB), 신주인수권부사채(BW) 등 법정 제도 |

## 📊 [한국 법체계에서의 SAFE 분석]

### 1️⃣ **법적 성격 논란**

```
🔹 현행법상 분류 문제:
   - 상법상 '주식'도, '사채'도 아닌 특수계약
   - 금융위 해석: "지분전환권이 있는 금전소비대차 유사 계약"
   
🔹 등기/공시 의무:
   - 등기부에 직접 기재 불가 (주식이 아니므로)
   - 주주명부 별도 관리 필요
```

### 2️⃣ **벤처투자법 vs. 일반 계약**

| 항목 | 벤처투자법상 투자 | SAFE 계약 |
|------|------------------|-----------|
| 법적 근거 | 「벤처투자 촉진에 관한 법률」 제2조 | 당사자 간 자유 계약 |
| 세제 혜택 | 투자자 소득공제/과세이연 가능 | **불가능** (금융투자상품 아님) |
| 보호 제도 | 벤처투자조합 등록 시 금융위 감독 | **자율 계약** (분쟁 시 민사소송) |

⚠️ **중요**: 엔젤투자자가 '소득공제'를 기대한다면 **벤처투자법상 인정 방식**이어야 합니다.

## 🚀 [Step-by-Step 실행 가이드]

### **[1단계] 투자자 의도 확인 (즉시)**
```
✓ 투자자에게 확인할 사항:
  1. 소득공제 필요 여부 (필요 시 CB/BW 전환 협의)
  2. SAFE 표준 조건 vs. 커스터마이징 의향
  3. 예상 전환 시점 (Series A 라운드 등)
```

### **[2단계] 법률 검토 (1주일 이내)**
```
📌 필수 확인 사항:
  ① 전환 가격 산정 방식 명확화
     - 밸류에이션 캡: 예) 50억 원 상한
     - 할인율: 예) Series A 가격의 20% 할인
  
  ② 전환 트리거 조건 구체화
     - "다음 프라이스드 라운드(Next Priced Round)" 정의
     - M&A 시 처리 방안 (주식 전환 or 현금 상환)
  
  ③ 이사회 의결정족수 및 거부권 설정
     - 6인 규모이므로 소수주주 권한 명시 중요
```

**💡 실무 팁**: 법무법인 선정 시 **스타트업 투자 전문** 로펌을 선택하십시오.  
추천: 법무법인 광장/세종/율촌의 '벤처/스타트업 그룹' 또는 '스타트업 특화 로펌' (린앤킴, 에이앤코 등)

### **[3단계] 세무사 협의 (병행)**
```
🔸 핵심 세무 이슈:
  1. 투자금 2억 원의 회계 처리
     ➜ "장기미지급금" or "기타자본잉여금" 처리 필요
  
  2. 전환 시점의 증여세 리스크
     ➜ 저가 전환 시 '이익 공여'로 간주 가능성
  
  3. 투자자의 양도소득세 과세 시점
     ➜ 전환 시점 vs. 실제 매각 시점 논란
```

### **[4단계] 대안 비교표 작성 (투자자 제시용)**

| 비교 항목 | SAFE | 전환사채(CB) | 신주인수권(BW) |
|----------|------|-------------|---------------|
| 법적 안정성 | △ (판례 부족) | ◎ (상법상 명시) | ◎ |
| 투자자 세제혜택 | ✗ | ◎ (조건 충족 시) | ◎ |
| 발행 절차 | 간편 (계약서만) | 복잡 (등기/공시) | 중간 |
| 이자 부담 | 없음 | **있음** (연 1~3%) | 없음 |
| 추천 상황 | 빠른 실행 우선 | 기관투자자 참여 | 소액 다수 투자 |

## ⚠️ [리스크 체크리스트]

### 🚨 **반드시 계약서에 명시해야 할 조항**

1. **전환 불가 시 처리 방안**
   ```
   예시: "투자 후 3년 내 적격 투자 유치 실패 시,
         연 5% 이자를 포함한 원금 상환 의무 발생"
   ```

2. **희석 방지 조항(Anti-dilution)**
   ```
   - Full Ratchet vs. Weighted Average 중 선택
   - 의료기기 스타트업 특성상 장기 개발 기간 고려 필요
   ```

3. **정보 제공 의무**
   ```
   분기별 재무제표, 임상시험 진행 상황 등 공유 범위 설정
   ```

4. **우선협상권 및 동반매도권**
   ```
   - 투자자의 Tag-Along 권리 범위
   - Exit 시 회수 우선순위 (1x, 2x 등)
   ```

### 🔒 **의료기기 스타트업 특수성 고려사항**

- **식약처 인허가 지연 리스크**: 전환 시점을 '인허가 획득 후'로 설정하는 조항 검토
- **임상데이터 소유권**: SAFE 투자자의 기술 IP 권리 명시 (의학 박사 CEO의 개인 IP와 분리)
- **의료법상 제한**: 병원 영업 시 비의료인 지분 제한 규정 저촉 여부 확인

## 🎁 [정부 지원 활용 전략]

   - 1억 원 규모 사업화 지원 (무상)
   - SAFE 투자와 병행 가능
