# Claude 4.5 Sonnet  응답 (1/3회차)

## 응답 내용
# [노무/인사 전문가 답변]

**'수습기간 = 자유롭게 해고 가능'이라는 인식은 매우 위험합니다.**  
귀사의 상황을 노무 전문가 관점에서 정확히 진단하고, 법적 리스크 없이 처리하는 방법을 안내드리겠습니다.

---

## 📋 **상황 진단**

**기업 프로필 분석:**
- 소규모 콘텐츠 기업 (총 7명)
- 해당 직원: 에디터 직군, 근무 2개월 차
- 문제: 업무 능력 부족 + 팀 협업 문제

**핵심 법적 쟁점:**
귀사가 **"내일부터 나오지 마라"고 통보할 경우, 부당해고로 소송당할 확률이 매우 높습니다.** 수습기간이라도 근로기준법상 최소한의 절차와 정당한 사유가 필요합니다.

---

## ⚠️ **즉시 해고 시 발생 가능한 리스크**

| 리스크 유형 | 구체적 내용 | 예상 비용 |
|------------|------------|----------|
| **부당해고 소송** | 근로자가 노동위원회 구제신청 → 승소 시 복직 or 임금 보상 명령 | 평균 3~12개월치 임금 |
| **퇴직금 미지급** | 1년 미만이라도 계속근로 기간에 따라 퇴직금 발생 가능 | 약 월급의 2/12 |
| **예고 없는 해고** | 30일 전 예고 또는 30일분 평균임금 지급 의무 | 1개월치 급여 |
| **평판 리스크** | 불만족 직원의 SNS/잡플래닛 후기로 채용 브랜드 훼손 | 측정 불가 |

**→ 총 예상 손실: 최소 500만 원 + 법률비용 + 시간 소모**

---

## ✅ **합법적이고 안전한 해고 절차 (Step-by-Step)**

### **Phase 1: 즉시 조치 (오늘~3일 이내)**

**1단계: 객관적 증거 확보**
```
[지금 당장 해야 할 일]
□ 업무 능력 부족 사례 문서화
  - 원고 퀄리티 문제: 수정 요청 이메일, 슬랙 대화 캡처
  - 피드백 불수용 사례: 회의록, 1:1 면담 기록
  
□ 동료 증언 확보 (선택사항)
  - 편집장이 다른 에디터 4명에게 협업 상황 서면 질의
  
□ 수습기간 평가 기준 재확인
  - 채용 공고/근로계약서에 명시된 평가 기준 확인
  - 없다면? → 일반적인 '업무 수행 능력' 기준 적용 가능
```

**2단계: 최종 개선 기회 부여 (필수)**
```
[면담 진행 방법]
✓ 일시: 2~3일 내 공식 면담 일정 잡기
✓ 참석자: 편집장(또는 대표) + 해당 직원 (증인 1명 동석 권장)
✓ 내용:
  "○○님의 수습기간이 곧 종료됩니다. 그러나 다음 3가지 부분에서
   개선이 필요하다고 판단됩니다:
   1) 원고 오류율 (구체적 수치 제시)
   2) 피드백 수용 태도
   3) 팀 협업 문제
   
   최종 1주일간 개선 기회를 드립니다. 구체적 개선 목표는 [OOO]입니다."

✓ 기록: 면담 내용 서면화 → 직원에게 서명 받기 (거부 시 증인 확인)
```

---

### **Phase 2: 최종 결정 (7~14일 이내)**

**3단계: 수습 평가 및 해고 결정**

**[합법적 해고 통보 방법]**
```
1. 통보 시점: 수습기간 종료 최소 7일 전
2. 통보 방식: 서면 (이메일 + 내용증명 권장)
3. 통보 내용 예시:

---
제목: 수습기간 평가 결과 및 근로계약 종료 안내

○○○님께,

2024년 X월 X일부터 시작된 수습기간 평가 결과,
다음의 사유로 정규직 전환이 어렵다고 판단되어
근로계약을 종료함을 알려드립니다.

[구체적 사유]
1. 원고 오류율: 평균 15% (목표 5% 이하)
2. 피드백 수용: 3회 이상 개선 요청에도 불구 미개선
3. 팀 협업: 동료 에디터와의 업무 조율 불가

최종 근무일: 2024년 X월 X일
퇴직금 및 미지급 임금: X월 X일 지급 예정

---
```

**4단계: 금전 정리 (해고 당일)**
```
□ 지급 항목 체크리스트
  ✓ 마지막 월 급여 (일할 계산)
  ✓ 미사용 연차 수당
  ✓ 퇴직금 (계속근로 1개월 이상 시 발생)
  ✓ 해고예고수당 (30일 전 통보 못했을 경우)
  
□ 서류 전달
  ✓ 근로소득원천징수영수증
  ✓ 퇴직증명서
  ✓ 고용·산재보험 자격상실신고서
```

## 🛡️ **리스크 최소화 체크리스트**

| 점검 항목 | 완료 여부 |
|----------|----------|
| 수습기간이 근로계약서에 명시되어 있는가? | □ |
| 해고 사유가 객관적으로 입증 가능한가? | □ |
| 최소 1회 이상 개선 기회를 부여했는가? | □ |
| 해고 예고를 30일 전에 했는가? (아니면 수당 지급) | □ |
| 모든 대화 내용이 문서화되어 있는가? | □ |
| 퇴직금/미지급 임금을 정확히 계산했는가? | □ |

**→ 위 6가지가 모두 충족되어야 법적 분쟁 가능성 5% 이하**

## 💡 **실전 팁: 수습기간 활용법**

**향후 채용 시 개선 방안:**
```
1. 명확한 평가 기준 사전 고지
   - 근로계약서에 "수습기간 중 [OOO] 기준 미달 시 계약 종료" 명시
   
2. 단계별 피드백 시스템
   - 1개월 차: 중간 평가 면담
   - 2개월 차: 최종 경고 및 개선 기회
   - 3개월 차: 최종 평가
   
3. 수습 기간 축소 고려
   - 콘텐츠 에디터의 경우 1~2개월로 단축 가능
   - 조기 정규직 전환 vs 조기 계약 종료 판단 빠르게
```

## 🚨 **절대 하지 말아야 할 것**

❌ **"내일부터 나오지 마" 구두 통보**  
→ 해고 예고 의무 위반 + 증거 없음 = 소송 패소 확률 90%

❌ **"팀 분위기 해친다"는 추상적 사유만 제시**  
→ 주관적 평가는 부당해고 인정될 가능성 높음

❌ **퇴직금/수당 미지급 후 "나중에 계산"**  
→ 체불 신고 시 사업주 형사처벌 대상

## 📞 **후속 질문 제안**

귀사의 성장을 위해 추가로 고민하면 좋을 지점들입니다:

1. **"우리 회사 근로계약서가 법적으로 완벽한가요?"**

   → 포트폴리오 과제, 유급 인턴십 등 대안 제시 가능
