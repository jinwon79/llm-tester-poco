## 🔍 핵심 솔루션: 합법적 수습 해지 절차

### ❌ 절대 금지 사항
```
"내일부터 나오지 마세요" → 부당해고 + 해고예고 위반
예상 손실: 평균임금 30일분 + 부당해고 구제신청 시 임금 소급
```

### ✅ 합법적 4단계 프로세스

| 단계 | 조치 내용 | 법적 근거 | 기한 |
|------|-----------|-----------|------|
| **1. 객관적 평가** | 업무 능력 평가표 작성 (구체적 사례 기록) | 근로기준법 제23조 | 즉시 |
| **2. 사전 경고** | 1차 구두 경고 + 2차 서면 경고 (개선 기한 명시) | 정당한 이유 입증 | 1~2주 |
| **3. 해고 예고** | 30일 전 서면 통지 OR 30일분 평균임금 지급 | 제26조 해고예고 | 필수 |
| **4. 해고 사유서** | 구체적 사유 기재한 서면 교부 | 제27조 | 해고 시 |

## 📝 Step-by-Step 실행 가이드

### **🚨 지금 당장 (D-Day ~ D+3)**

**1. 증거 확보 (필수)**
```
✓ 업무 지시 이메일 · 메신저 기록
✓ 원고 피드백 이력 (날짜별 정리)
✓ 품질 미달 사례 (구체적 오류 건수)
✓ 팀원 면담 기록 (익명 가능)
```

**2. 평가표 작성 샘플**
```
[업무 수행 능력 평가서]
평가 기간: 2025.XX.XX ~ 2025.XX.XX
평가 항목:
1. 원고 작성 능력: 하 (기준 대비 60% 수준)
   - 사례: 제출 원고 10건 중 8건 전면 재작성 필요
2. 피드백 수용도: 불량
   - 사례: 동일 오류 3회 반복 (날짜 기재)
3. 팀워크: 불량
   - 사례: 회의 중 의견 충돌 3회 (00팀장 증언)
```

### **📅 단기 (1~2주)**

**3. 개선 기회 제공 (법적 필수 절차)**
```
[서면 경고장 예시]
수신: 에디터 OOO
발신: (주)콘텐츠 큐레이터 편집장

귀하의 업무 수행 능력이 회사 기준에 미달하여 다음과 같이 
개선을 요구합니다.

개선 요구 사항:
1. 원고 1차 제출 시 오류율 30% 이하 유지
2. 피드백 반영률 90% 이상 달성
개선 기한: 2025.X.X까지 (14일)

미개선 시 근로계약 해지될 수 있음을 알려드립니다.
```

**4. 개선 평가 실시**
- 기한 내 동일 기준으로 재평가
- 개선 없을 시 다음 단계 진행

### **📅 중기 (2~4주)**

**5. 해고 예고 (두 가지 옵션)**

| 옵션 A | 옵션 B |
|--------|--------|
| 30일 전 통지 (계속 근무) | 즉시 통지 + 30일분 급여 지급 |
| 추가 급여 부담 없음 | 즉각 분리 가능 |
| **추천: 분위기 악화 우려 시** | **추천: 즉시 분리 필요 시** |

**옵션 B 계산 예시 (월급 250만원 가정)**
```
평균임금 = 최근 3개월 급여 총액 / 총 일수
해고예고수당 = 평균임금 × 30일
= 약 250만원
```

**6. 해고통지서 작성**
```
[근로계약 해지 통지서]

근로기준법 제23조 및 제26조에 따라 다음과 같이 통지합니다.

해고 사유:
1. 수습 기간 중 업무 능력 현저히 부족 (평가서 첨부)
2. 반복 경고에도 개선 의지 없음 (경고장 2회 발송)
3. 팀 협업 불가로 업무 지장 초래

해고 예정일: 2025.X.X
해고예고수당: 30일분 지급 (OR 30일 후 해고)

※ 이의 있을 시 14일 이내 노동위원회 구제신청 가능
```
