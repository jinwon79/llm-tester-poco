# Claude 4.5 Sonnet  응답 (2/3회차)

## 응답 내용
# 노무/인사 분야 전문가로서 답변드리겠습니다

귀사의 상황은 **수습기간 해고의 적법성**과 **조직 운영 리스크**를 동시에 고려해야 하는 민감한 사안입니다. 즉시 해고는 법적 분쟁으로 이어질 수 있어 신중한 절차가 필요합니다.

---

## 1️⃣ 상황 진단

### 파악된 정보
- 수습기간 3개월 중 2개월 경과
- 업무 능력 미달 + 피드백 수용 불가
- 팀 규모 7명의 소규모 조직 (영향도 큼)
- 콘텐츠 품질이 직접적으로 수익과 연결되는 구조

### 추가 확인 필요 사항
더 정확한 컨설팅을 위해 다음 정보가 필요합니다:
- ✓ 근로계약서에 **수습기간 명시** 여부
- ✓ 취업규칙 신고 여부 (상시근로자 5인 이상 의무)
- ✓ 구체적 업무 미달 **증빙 자료** 보유 여부 (피드백 이력, 원고 샘플 등)
- ✓ 수습 평가 기준 사전 고지 여부

---

## 2️⃣ 핵심 솔루션: "즉시 해고"는 불가능합니다

### ❌ 절대 하면 안 되는 행동
> **"내일부터 나오지 마세요"** 통보 시 발생 가능한 법적 리스크:

| 리스크 | 내용 | 예상 비용 |
|--------|------|----------|
| **부당해고 판정** | 정당한 사유 + 적법한 절차 미준수 시 | 평균임금 3개월분 |
| **민사소송** | 위자료 청구 (팀 규모 작아 증거 확보 쉬움) | 500만~2,000만원 |
| **근로기준법 위반** | 30일 전 해고예고 또는 예고수당 미지급 시 | 2년 이하 징역/1천만원 이하 벌금 |

### ✅ 수습기간 해고의 법적 요건

근로기준법상 **수습 중 해고도 정당한 사유와 절차 필수**:

#### (1) 정당한 사유 입증 책임 (사용자)
- 단순 "능력 부족"만으로는 불충분
- **객관적 평가 기준 + 반복적 피드백 + 개선 기회 부여** 증명 필요

#### (2) 적법한 절차
```
수습 중이어도 30일 전 해고예고 원칙
→ 단, 예고수당(평균임금 30일분) 지급 시 즉시 해고 가능
```

## 3️⃣ Step-by-Step 실행 가이드

### 🔹 1단계: 긴급 증빙 확보 (오늘~내일)

#### 해야 할 일
1. **수습 평가 체크리스트 작성**
   ```
   예시:
   - 원고 제출 기한 준수율: 40% (5회 중 2회만 준수)
   - 수정 요청 횟수: 평균 4.2회 (팀 평균 1.5회)
   - 구독자 피드백 점수: 2.3/5.0 (허용 기준 3.5 이상)
   ```

2. **피드백 이력 문서화**
   - 이메일, 메신저 대화 내용 캡처
   - 편집 회의록에서 해당 직원 언급 부분 발췌

3. **동료 면담 기록**
   - "협업 시 어려움" 등 객관적 사실 중심 정리
   - (주의: 인격 모독적 표현 절대 금지)

---

### 🔹 2단계: 최종 경고 및 개선 기회 부여 (D+2~3일)

#### 면담 프로세스
```
[면담 일시] 2025년 X월 X일 14:00
[참석자] 편집장 + 해당 직원 (제3자 배석 권장)
[내용]
"지난 2개월간의 업무 성과를 검토한 결과,
수습 평가 기준(원고 품질 △△, 마감 준수율 ○○)에 미달하였습니다.
수습 종료일(X월 X일)까지 다음 사항 개선 필요합니다:
① 원고 1차 제출 시 수정 요청 2회 이하
② 마감일 준수율 90% 이상
③ 구독자 만족도 3.5점 이상

개선되지 않을 경우 수습 종료 시점에 근로계약 해지될 수 있음을 알려드립니다."
```

#### 필수 조치
- 면담 내용 **서면 작성 → 직원 서명** 받기
- 서명 거부 시 "내용증명 발송" 또는 "배석자 확인서" 작성

---

### 🔹 3단계: 합법적 해고 절차 (수습 종료일 기준)

#### Case A: 개선 없이 수습 종료일 도래 시
```
[해고예고]
수습 종료 30일 전까지 서면 통지
   예) 수습 종료일이 1월 31일 → 1월 1일까지 통지

[대안] 즉시 해고 원할 시
→ 평균임금 30일분 "해고예고수당" 지급
   계산식: (최근 3개월 임금 총액 ÷ 총 일수) × 30일
```

#### Case B: 명백한 중대 과실 발생 시 (예외)
- 구독자 정보 유출, 고의적 업무 방해 등
- 이 경우에도 **즉각 징계위원회 + 소명 기회** 필수

## 6️⃣ 조직 운영 관점 추가 제언

### 재발 방지 시스템 구축
1. **채용 단계 강화**
   - 포트폴리오 검증 강화
   - 테스트 원고 작성 (유료 과제)

2. **온보딩 체계화**
   ```
   1주차: 편집 가이드 교육 + 멘토 배정
   2주차: 첫 원고 제출 → 즉시 피드백
   4주차: 중간 평가 + 개선 방향 논의
   8주차: 최종 평가 → 정규직 전환 여부 결정
   ```

3. **계약서 템플릿 정비**
   - "수습 평가 기준" 명시 조항 추가
   - 노무사 검토 필수 (비용: 30~50만원)
