## 📊 [상황 진단]

### 파악된 핵심 이슈
✅ **기술**: GPT-4 API 의존 → 비용 부담 + 데이터 주권 리스크  
✅ **비즈니스**: B2B(학원) 확장 시 데이터 보안 요구 증가 예상  
✅ **조직**: AI 인력 2명 → 자체 모델 운영 가능 여부가 관건  

### 추가로 필요한 정보
더 정확한 로드맵 수립을 위해 아래 정보를 알려주시면 최적화된 전략이 가능합니다:
- **월 API 비용** (GPT-4 호출량)
- **보유 학습 데이터 규모** (문제 수, 오답 케이스 수)
- **목표 고객 규모** (12개월 내 예상 MAU)
- **자금 여력** (GPU 서버 투자 가능 예산)

## 🎯 [핵심 솔루션] 단계별 전환 전략

| 구분 | OpenAI API | 오픈소스 자체 모델 | **하이브리드(권장)** |
|------|-----------|----------------|----------------|
| **초기 비용** | 낮음 (종량제) | 높음 (인프라+인건비) | **중간 (단계적 투자)** |
| **장기 비용** | 높음 (스케일 시 급증) | 낮음 (고정비화) | **최적화 가능** |
| **데이터 주권** | ❌ OpenAI 정책 의존 | ✅ 완전 통제 | **✅ 핵심 기능만 내재화** |
| **성능 커스터마이징** | 제한적 (프롬프트 엔지니어링) | 완전 자유 | **핵심 도메인 특화** |
| **운영 난이도** | 쉬움 | 어려움 (MLOps 필수) | **점진적 내재화** |

### 💡 권장 접근: **3단계 진화 모델**

```
[1단계] API + 파인튜닝 병행 (현재~6개월)
   ↓
[2단계] 하이브리드 운영 (7~18개월)
   ↓
[3단계] 자체 모델 중심 (19개월~)
```

## 🛠️ [Step-by-Step 실행 가이드]

### **PHASE 1: 준비 및 검증 (0~6개월)**

#### 1️⃣ 비용 구조 분석 (1주 이내)
```python
# 월 API 비용 계산식
월 비용 = (일 평균 문제 생성 수) × (토큰/문제) × (GPT-4 단가)

# 손익분기점 계산
자체 모델 전환 타당성 = 월 API 비용 > $3,000 + (자체 인프라 월 비용 $1,500)
```
**Action**: 지난 3개월 OpenAI 청구서 분석 → 문제당 원가 산출

#### 2️⃣ 데이터 준비 (병행 진행)
- **필수**: 최소 10,000개 이상의 **고품질 문제-정답-해설 세트**
- **라벨링**: 난이도, 단원, 오답 패턴 메타데이터 추가
- **정부 지원 활용**: 
  - [AI 바우처 지원사업](https://www.msit.go.kr) → 데이터 가공비 최대 7,000만원 지원
  - [AI 학습용 데이터 구축사업](https://aihub.or.kr) → 무료 교육 데이터셋 활용 가능

#### 3️⃣ PoC 실험 (2~3개월)
**실험 설계**:
| 모델 | 용도 | 예상 성능 |
|------|------|----------|
| GPT-4 API | 복잡한 서술형 문제 | Baseline |
| Llama 3.1 8B (LoRA 튜닝) | 객관식 유사문제 생성 | 80~85% 수준 목표 |

**기술 스택**:
```
- 모델: Llama 3.1 8B (상업적 이용 가능)
- 파인튜닝: LoRA (파라미터 효율적)
- 인프라: AWS SageMaker or Modal Labs (초기 서버리스 권장)
- 예산: 월 $300~500 수준에서 테스트 가능
```

---

### **PHASE 2: 하이브리드 운영 (7~18개월)**

#### 1️⃣ 기능별 모델 분리
```
[자체 모델 담당]
✓ 객관식 유사 문제 생성 (90% 트래픽)
✓ 난이도 자동 분류
✓ 오답 패턴 분석

[OpenAI API 담당]
✓ 서술형 문제 해설 생성 (10% 트래픽)
✓ 신규 과목 콜드 스타트
```
**효과**: API 비용 60~70% 절감 + 핵심 기술 내재화

#### 2️⃣ 온프레미스 vs 클라우드 선택
| 항목 | 클라우드 (AWS/GCP) | 온프레미스 (자체 서버) |
|------|----------------|------------------|
| **초기 투자** | 거의 없음 | GPU 서버 5,000만원~ |
| **월 운영비** | $1,000~3,000 | 전기세+유지보수 50만원 |
| **확장성** | 즉시 가능 | 제한적 |
| **권장 시점** | **현재~MAU 10만 이하** | MAU 50만 이상 |

**현재 단계 권장**: AWS SageMaker Inference 활용 (오토스케일링)

---

### **PHASE 3: 완전 자체화 (19개월 이후)**

#### 조건 충족 시 전환
- ✅ MAU 50만 이상 / 월 API 비용 $10,000 초과
- ✅ 자체 모델 정확도 GPT-4 대비 95% 이상
- ✅ MLOps 전담 인력 확보 (DevOps 1명 + AI 엔지니어 1명)

#### 인프라 설계
```
[추천 구성]
- GPU: NVIDIA A100 (40GB) × 2대
- 서버: On-premise or AWS EC2 p4d 인스턴스
- 배포: TensorRT 최적화 + Kubernetes
- 모니터링: Prometheus + Grafana
```
