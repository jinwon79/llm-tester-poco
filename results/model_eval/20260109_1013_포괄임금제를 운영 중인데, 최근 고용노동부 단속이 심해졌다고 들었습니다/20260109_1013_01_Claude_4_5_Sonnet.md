## 🛠️ [Step-by-Step 실행 가이드]

### 📌 **즉시 조치 (1주 내)**

**1단계: 긴급 자가진단**
```
[ ] 최근 3개월 전 직원 근로시간 기록 확인
[ ] 계약서상 '포괄수당' 항목별 금액 명시 여부 체크
[ ] 야간근무(22시~06시) 발생 빈도 집계
[ ] 통상임금 × 1.5 × 실제 연장시간 계산 → 현재 지급액과 비교
```

**2단계: 법률 검토 (노무사 상담 필수)**
- **추천 경로**: 
  - 고용노동부 무료 노무상담 (☎1350)
  - 중소벤처기업진흥공단 '노무관리 컨설팅' 지원사업 (비용 70% 지원)
  - 대한노무법인협회 초기 상담 (30분 무료)

### 📊 **중기 개선안 (1~3개월)**

#### ✅ **Option 1: 합법적 포괄임금제 재설계**

**개선된 계약서 예시**
```
[월 급여 4,000,000원 구성]
- 기본급: 2,500,000원 (통상임금)
- 직책수당: 300,000원
- 고정 연장수당: 900,000원 (월 52시간 기준)
  → 2,500,000÷209h×1.5×52h = 933,014원
- 야간수당: 별도 실비 지급 (시간당 17,942원)
```

**핵심 포인트:**
- 기본급 비중 60% 이상 확보
- 포괄수당 산정 근거를 계약서에 명시
- **야간수당은 '별도 지급' 필수 명시**

---

#### ✅ **Option 2: 교대근무제 도입 (근본적 해결)**

| 운영 방식 | 장점 | 단점 |
|----------|-----|-----|
| **3조 3교대** | 야간수당 최소화, 직원 건강 관리 | 인력 25→30명 증원 필요 |
| **주·야 2교대** | 현 인원 유지 가능 | 야간수당 부담 지속 |

**정부 지원 활용:**
- 고용부 '교대제 개편 지원사업' (컨설팅 + 인건비 지원)
- 신청: 고용센터 직접 방문 또는 worktogether.or.kr

### 📈 **장기 전략 (3~6개월)**

**1. 임금피크제 + 성과급 전환**
- 고정 연장수당 축소 → 분기별 성과급으로 전환
- 게임 매출 연동 인센티브 설계
- 세무상 비용 처리 최적화 (복리후생비 vs 급여)

**2. 재택/유연근무 병행**
- 비긴급 업무는 재택 전환 (야간 출근 불필요)
- '선택적 근로시간제' 도입 검토 (근로기준법 제52조)

**3. HR 시스템 고도화**
- 근태관리 솔루션 도입 (예: 잡플래닛, 플렉스타임)
- AI 기반 CS 자동화 → 야간 인력 최소화

⚠️ **본 답변은 일반적인 가이드이며, 개별 사안의 법률 자문을 대체할 수 없습니다.**

- 실제 계약서 검토 및 과거 근로시간 기록 분석은 **반드시 공인노무사 또는 노동 전문 변호사**와 진행하셔야 합니다.
- 이미 직원의 진정이나 고소가 제기된 경우, 즉시 법률 대리인을 선임하십시오.
- 포괄임금제 재설계 시 기존 직원의 '불이익 변경' 문제가 발생할 수 있으므로, 동의서 징구 절차가 필수입니다.

## 🎯 [후속 질문 제안]

귀사의 지속 성장을 위해 추가로 고민해 보시면 좋을 주제들입니다:

1. **"게임 업계 특성에 맞는 탄력적 근로시간제 설계 방법이 궁금합니다"**
   → 성수기/비수기 대응 전략 제시 가능

2. **"직원 25명 규모에서 HR 담당자를 별도로 채용해야 할까요?"**
   → 아웃소싱 vs 정규직 비용 비교 분석 제공

3. **"연장근무 대신 인력을 늘리면 세금 혜택이 있나요?"**
   → 청년 고용 세액공제, 고용증대 세액공제 연계 가능

4. **"주 52시간 위반 없이 긴급 점검을 할 수 있는 방법은?"**
   → 비상대응 체계 및 특별연장근로 인가 절차 안내
