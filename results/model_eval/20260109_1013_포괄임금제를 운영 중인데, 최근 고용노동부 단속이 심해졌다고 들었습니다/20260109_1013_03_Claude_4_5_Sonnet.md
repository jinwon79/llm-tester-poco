## 응답 내용
# 노무/인사 분야 전문가로서 답변드리겠습니다

귀사의 상황은 **고위험 노무 이슈**에 해당합니다. 포괄임금제는 합법적 제도이나, 게임 업계의 특성상 '24시간 모니터링'과 '잦은 연장근무'가 있다면 **현재 운영 방식이 위법일 가능성이 매우 높습니다**.

---

## 📊 상황 진단

### 현재 파악된 정보
- ✅ 업종: 게임 퍼블리싱/운영 (야간·휴일 근무 빈번)
- ✅ 인원: 운영지원/CS팀 25명
- ✅ 계약서: "연장수당 포함" 문구 명시
- ⚠️ 24시간 모니터링 체제 운영 중

### 추가 확인 필요 사항
더 정확한 컨설팅을 위해 다음 정보가 필요합니다:
1. **근로계약서상 기본급과 포괄수당 금액 분리 여부**
2. **실제 월평균 연장/야간/휴일 근무 시간**
3. **교대제(2교대/3교대) 운영 여부**
4. **근로시간 기록 방식** (PC-ON/OFF, 출퇴근 카드 등)

---

## ⚖️ 법적 판단 (핵심)

### ❌ 위법 가능성이 높은 경우
귀사가 다음에 해당한다면 **즉시 시정**이 필요합니다:

| 위법 유형 | 귀사 적용 가능성 | 법적 리스크 |
|---------|--------------|-----------|
| **1. 야간수당(50%) 미지급** | 🔴 **매우 높음** | 3년치 미지급 임금 + 지연이자 |
| 2. 실제 근로시간이 계약서상 예정보다 20% 초과 | 🟡 확인 필요 | 포괄임금제 무효 판정 |
| 3. 기본급과 수당 구분 없이 "총액 ○○만원"으로만 표기 | 🟡 확인 필요 | 근로계약 무효 소지 |
| 4. 52시간 상한 위반 (주 12시간 연장 초과) | 🔴 **매우 높음** | 2년 이하 징역 또는 2천만원 이하 벌금 |

### ✅ 포괄임금제가 합법인 조건 (대법원 판례 기준)
```
1. 근로시간 산정이 어려운 업무 (×게임 CS는 시간 측정 가능)
2. 기본급과 수당이 명확히 구분되어 있을 것
3. 실제 근로에 대한 정당한 보상이 이루어질 것
4. 통상임금보다 불리하지 않을 것
```

**👉 귀사의 24시간 모니터링 업무는 "시간 산정 가능"하므로 포괄임금제 적용 자체가 부적합할 가능성 높음**

## 🎯 핵심 솔루션

### 🚨 **야간 수당 관련 핵심 답변**
> **"연장수당 포함" 문구만으로는 야간수당(22시~06시, 50% 가산)을 면제받을 수 없습니다.**

- **근로기준법 제56조**: 야간근로(22~06시)는 **통상임금의 50%를 가산**해야 함
- **포괄임금 계약서에 "야간수당 포함"이라는 명시적 문구 + 해당 금액 산정 근거**가 없다면 **미지급으로 간주**

## 📋 Step-by-Step 실행 가이드

### 📌 **STEP 1. 즉시 조치 (이번 주 내)**

#### 1-1. 긴급 자가 진단 체크리스트
```
□ 최근 3개월 직원별 실제 근로시간 집계
□ 야간근무(22~06시) 일수 및 시간 확인
□ 현재 계약서상 포괄수당 금액이 실제 연장/야간 수당보다 많은지 역산
□ 주 52시간(연장 12시간 포함) 초과 직원 여부 확인
```

#### 1-2. 노무사 긴급 자문 (필수)
- **자문 내용**: 현재 계약서 및 실제 근로시간 기록 검토
- **비용**: 초기 자문 20~50만원 (노동부 조사 대비 필수 투자)
- **추천 경로**: 
  - 대한노무법인 ([www.daehanhr.com](http://www.daehanhr.com))
  - 지역 노무사회 무료 상담 (1588-0075)

---

### 📌 **STEP 2. 중기 대책 (1~2개월)**

#### 2-1. 근로계약서 전면 개정
```
[개정 필수 항목]
1. 기본급과 각종 수당(연장/야간/휴일) 명확히 분리 표기
2. 예정 근로시간과 포괄수당 산정 근거 명시
   예: "월 평균 연장 20시간, 야간 10시간 예정, 해당 수당 ○○원"
3. 실제 근로가 예정보다 20% 초과 시 추가 지급 조항 삽입
```

#### 2-2. 근로시간 관리 시스템 도입
| 방법 | 비용 | 효과 |
|-----|------|------|
| **PC-ON/OFF 자동 기록** | 월 5만원~ | 실시간 근로시간 추적 |
| 플렉스타임제 도입 | 제도 설계 비용 | 52시간 준수 용이 |
| 교대제 개편 (3조 3교대) | 인력 증원 필요 | 야간수당 절감 + 근로조건 개선 |

---

### 📌 **STEP 3. 장기 전략 (3개월~)**

#### 3-1. 인력 운영 최적화
```
[현재 문제점]
- 25명으로 24시간 커버 → 1인당 과부하

[개선안]
Option A. 3조 3교대 전환 (인원 유지)
  - 주간조(09~18시), 오후조(15~24시), 야간조(22~07시)
  - 야간수당 발생하나 근로시간 준수 가능

Option B. CS 아웃소싱 병행
  - 야간/새벽 시간대만 외주 전환 (월 300~500만원)
  - 정규직 야간수당보다 비용 절감 가능
```

#### 3-2. 정부 지원 활용
- **근로시간 단축 지원금**: 교대제 개편 시 인건비 지원 (1인당 월 40만원)
  - 신청처: 고용노동부 [www.work.go.kr](http://www.work.go.kr)
- **청년 추가 고용 지원**: 야간조 인력 충원 시 1인당 최대 960만원
  - K-Startup 고용연계 프로그램 연계 가능

## ⚠️ 리스크 체크리스트

### 🔴 **고위험 시나리오**
1. **근로감독 적발 시**
   - 3년치 미지급 임금 소급 (직원 1인당 평균 500~1,000만원)
   - 사업주 형사 처벌 (임금체불죄)
   - 명단 공개 및 정부 지원사업 제재

2. **직원 퇴사 후 진정/소송**
   - 최근 게임업계 집단 진정 사례 증가 추세
   - 승소 시 지연이자 연 20% 추가 부담

3. **투자 유치 시 노무 실사**
   - 미지급 임금이 '우발채무'로 분류되어 기업가치 하락

### 🟡 **주의 사항**
- 직원과 "합의서" 작성해도 법적 효력 없음 (근로기준법 강행규정)
- 포괄임금제 무효 판정 시 **전체 직원에 대해 소급 적용** 가능

## 💡 후속 질문 제안

귀사의 지속 가능한 성장을 위해 다음 사항도 검토해보시기 바랍니다:

1. **"현재 CS팀 이직률이 높지 않나요?"**
   - 야간 근무 과다 시 번아웃 → 채용/교육 비용 증가
   - 선제적 근로조건 개선이 장기적으로 비용 절감

2. **"게임 업데이트를 특정 시간대로 고정할 수 있나요?"**
   - 유지보수 시간 예측 가능 → 교대제 설계 용이

3. **"벤처 인증 시 노무 점검이 필수인데, 준비되셨나요?"**
   - 이노비즈/벤처 인증 심사 시 노무 실사 강화 추세
