## 📋 **1. 상황 진단 (Current State Analysis)**

### ✅ 강점 (Strengths)
- 핵심 알고리즘 국내 특허 등록 완료 (기본 방어선 확보)
- 기술 전문성 높은 팀 구성 (로봇공학 박사 3명 + AI 개발자 7명)
- 명확한 응용 분야 (드론 제조, 자율주행 물류)

### ⚠️ 리스크 (Risks)
- **단일 특허 의존도 높음** → 우회 설계(Design Around) 취약
- **글로벌 특허 미확보** 시 해외 시장 진입 시 침해 리스크
- **경쟁사 후발 주자의 개량 특허** 출원 가능성
- **응용 분야 확대에 따른 IP 공백** (방어 범위 불충분)

## 🎯 **2. 핵심 솔루션: 특허 장벽 전략 (Patent Barrier Strategy)**

### 📌 **전략 1: 특허 포트폴리오 피라미드 구축 (3-Layer Defense)**

```
[최상층] 원천 특허 (Core Patent)
    └─ 장애물 회피 알고리즘 핵심 로직

[중간층] 개량·응용 특허 (Improvement Patents) ← **에버그리닝 핵심**
    ├─ 저조도 환경 최적화 알고리즘
    ├─ 다중 드론 협업 회피 기술
    ├─ 실시간 학습 기반 경로 예측
    └─ 특정 센서(LiDAR, 비전) 통합 방법

[기저층] 응용 특허 (Application Patents)
    ├─ 물류 드론 전용 회피 시스템
    ├─ 실내 자율주행 로봇 적용 기술
    ├─ 긴급 상황 대응 알고리즘
    └─ UI/UX 및 사용자 인터페이스
```

**✅ 실행 포인트:**
- 원천 특허는 '방어의 핵', 중간층은 '경쟁사 우회 차단', 기저층은 '시장 세분화 독점'
- 중간층 특허를 **연 2~3건씩 지속 출원**하여 특허 만료 시기를 분산 (에버그리닝)

---

### 📌 **전략 2: 특허 맵핑 (Patent Landscape Analysis)**

#### **Step 1: 경쟁사 특허 역분석 (6개월 내 필수)**
| 작업 | 방법 | 예산 |
|------|------|------|
| 국내·해외 특허 검색 | KIPRIS, Google Patents, Espacenet | 무료~200만원 |
| 전문기관 의뢰 | 특허청 IP-NAVI, 민간 IP 조사 기관 | 500~1,000만원 |
| 자유실시기술(FTO) 분석 | 침해 위험 특허 식별 | 필수 |

**⚠️ 리스크:** 경쟁사가 이미 주변 기술을 선점했을 경우 교차 라이선스 또는 무효심판 필요

---

#### **Step 2: 기술 트리 기반 선제 출원 (Technology Tree Filing)**

귀사 알고리즘의 **변형 가능한 모든 시나리오**를 특허화:

```
[핵심 알고리즘]
    ├─ 입력 데이터 변형 (센서 종류별)
    ├─ 연산 방식 변형 (머신러닝 모델별)
    ├─ 출력 적용 변형 (드론/자동차/로봇)
    └─ 하드웨어 결합 (특정 칩셋 최적화)
```

**✅ 실행 액션:**
1. **기술 분해 워크숍** (팀 내부): 알고리즘의 모든 구성 요소를 분해하여 특허 가능성 평가
2. **우선순위 매트릭스 작성**:
   | 기술 | 시장 임팩트 | 우회 난이도 | 출원 우선순위 |
   |------|-------------|-------------|---------------|
   | 저조도 최적화 | 높음 | 높음 | ★★★ |
   | 다중 드론 협업 | 중간 | 중간 | ★★ |

### 📌 **전략 3: 해외 특허 전략 (Global IP Protection)**

#### **PCT 국제출원 로드맵**

| 단계 | 시기 | 타겟 국가 | 예산 (추정) |
|------|------|-----------|-------------|
| **1단계** | 즉시 | PCT 출원 (1건) | 500~800만원 |
| **2단계** | 18개월 내 | 미국, 중국, EU | 국가당 1,000~1,500만원 |
| **3단계** | 30개월 내 | 일본, 인도 (선택) | 국가당 800~1,200만원 |

**💡 Tip:** 
- **특허청 해외 출원 지원사업** 활용 시 최대 **50~70% 비용 절감**
  - K-Startup 해외진출 지원사업
  - 중소기업 지식재산 해외출원 비용 지원

### 📌 **전략 4: 영업비밀 병행 관리 (Trade Secret Hybrid)**

특허는 **20년 후 공개**되므로, 일부 핵심 노하우는 **영업비밀**로 관리:

| 구분 | 특허 출원 대상 | 영업비밀 관리 대상 |
|------|----------------|-------------------|
| **기준** | 경쟁사가 리버스 엔지니어링 가능 | 리버스 엔지니어링 불가능 |
| **예시** | 알고리즘 구조, 센서 통합 방법 | 학습 데이터셋, 최적화 파라미터 |
| **보호 기간** | 20년 (갱신 불가) | 무한 (관리 지속 시) |

**✅ 실행 체크리스트:**
- [ ] 핵심 개발자 **비밀유지계약(NDA)** 체결
- [ ] 소스코드 접근 권한 **역할 기반 제한** (RBAC)
- [ ] 퇴사자 **경업금지 약정** (법적 한계 검토 후 합리적 범위 설정)

## 🚀 **3. Step-by-Step 실행 가이드 (6개월 로드맵)**

### **[1~2개월] 기반 구축 단계**
- [ ] **Week 1-2**: 내부 기술 분해 워크숍 (전 개발팀 참여)
- [ ] **Week 3-4**: 경쟁사 특허 맵핑 (전문기관 의뢰 또는 자체 조사)
- [ ] **Month 2**: 우선 출원 대상 5~7건 선정 + 변리사 선임

### **[3~4개월] 출원 집중 단계**
- [ ] **국내 특허 3건 출원** (개량·응용 특허)
- [ ] **PCT 출원 1건** (핵심 알고리즘 글로벌 보호)
- [ ] **디자인/상표 출원** (브랜드 보호 병행)

### **[5~6개월] 방어 체계 완성**
- [ ] **영업비밀 관리 규정** 수립 (법무법인 자문)
- [ ] **특허 포트폴리오 관리 시스템** 구축 (엑셀 또는 IP 관리 SW)
- [ ] **분기별 특허 모니터링 체계** 구축 (경쟁사 신규 출원 추적)
