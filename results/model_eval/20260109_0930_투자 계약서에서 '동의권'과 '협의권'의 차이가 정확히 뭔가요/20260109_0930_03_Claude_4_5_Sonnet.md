# 법률/계약 전문가로서 답변드리겠습니다

**투자 계약서의 핵심 용어 해석 및 실무 리스크 분석**을 통해, 귀사의 경영권 보호 전략을 제시하겠습니다.

## [상황 진단]

✅ **파악된 정보**
- 프리 시리즈 A 단계 (초기 성장기)
- VC가 '사전 서면 동의권' 다수 요구
- 경영진 구성: 대표 + 핵심 인력 10명 (비교적 소규모)

⚠️ **추가 확인 필요 사항**
더 정확한 리스크 진단을 위해 다음 정보를 확인해주시면 좋습니다:
- VC 투자금액 및 지분율 (예: 20억 / 지분 25%)
- 현재 대표이사 지분율
- '동의권' 항목의 구체적 범위 (계약서 조항 번호)

---

## [핵심 솔루션] 동의권 vs 협의권 비교표

| 구분 | 동의권 (Consent Right) | 협의권 (Consultation Right) |
|------|----------------------|---------------------------|
| **법적 효력** | 투자자 동의 없으면 **실행 불가** | 투자자 의견 청취만 필요, **결정권은 회사** |
| **실무 예시** | 유상증자 → VC 반대 시 절대 진행 불가 | 유상증자 → VC 의견 듣되 이사회 의결로 진행 가능 |
| **거부권(Veto)** | ⭕ 있음 | ❌ 없음 |
| **위험도** | 🔴 **높음** (경영 마비 가능성) | 🟡 **중간** (소통 부담) |

## [Step-by-Step 실행 가이드]

### **1단계: 계약서 조항 분류 (지금 당장 할 일)**

계약서를 펼쳐놓고 다음과 같이 형광펜으로 표시하세요:

🔴 **빨간색 (절대 피해야 할 동의권)**
- "이사회 구성원 임면에 대한 사전 동의"
- "대표이사 해임 동의"
- "일상적 영업 활동(3,000만원 이하 지출 등)에 대한 동의"

🟡 **노란색 (협상 가능한 동의권)**
- "총 발행주식의 1% 이상 신규 발행 시 동의"
- "회사 자산 총액의 10% 초과 자산 매각 동의"
- "연간 예산 20% 초과 지출 시 동의"

🟢 **초록색 (수용 가능한 동의권)**
- "회사 청산/합병 시 동의"
- "정관 변경 시 동의"
- "3억원 이상 대출 시 동의"

### **2단계: 협상 전략 (이번 주 내 실행)**

```
[협상 시나리오 예시]

❌ 거부해야 할 조항:
"모든 신규 채용에 대한 VC 사전 동의"
→ 대안 제시: "연봉 1억 이상 임원급 채용 시에만 '협의권'"

✅ 조건부 수용 가능:
"5억원 이상 유상증자 시 동의"
→ 수정 요청: "10억원 이상으로 금액 상향 + 기존 주주 비례 참여 조건 명시"
```

**실제 협상 멘트**:
> "투자자님의 리스크 관리 필요성은 이해합니다. 다만 일상 경영까지 동의권이 적용되면 기술 개발 속도가 현저히 저하될 수 있어, **'분기별 보고 + 중대 사항 협의'** 방식으로 조정 가능할까요?"

### **3단계: 법률 검토 (계약 전 필수)**

✔️ **투자 계약서 전문 로펌 리스트** (제조 스타트업 특화)
- 법무법인 광장 (스타트업그룹)
- 법무법인 세종 (벤처 투자팀)
- 법무법인 율촌 (IP/Tech 그룹)

📞 **초기 상담 비용**: 계약서 검토 약 200~500만원 (협상 포함 시 별도)

## [리스크 체크리스트]

### 🚨 **즉각 위험 신호 (Red Flag)**

- [ ] "대표이사 전결 권한 50만원 미만"으로 제한
- [ ] "투자자 추천 이사의 단독 서면 동의 필요" 조항
- [ ] "동의 거부 시 투자금 즉시 회수" 조항
- [ ] "경쟁사 투자 금지" 없이 투자자의 포트폴리오 자유

### ⚠️ **장기 리스크 (운영 중 문제 가능)**

- [ ] **추가 투자 시 독소 조항**: "후속 라운드에서 VC 우선 참여권 + 동일 조건 보장"
- [ ] **Exit 제약**: "IPO/M&A 시 투자자 동의 + 최소 수익률 보장"
- [ ] **경영 간섭**: "월별 재무제표 제출 + VC 추천 CFO 필수 채용"

## [실전 대응 매뉴얼]

### **시나리오 1: VC가 조항 수정 거부 시**

```
[협상 카드]
1. 경쟁 투자자 제시: "B 투자사는 협의권으로 가능하다고 하셨습니다"
2. 단계적 완화 제안: "첫 1년은 동의권, 실적 목표 달성 시 협의권으로 전환"
3. 이사회 구성 조정: "독립 이사 1명 추가로 균형 확보"
```

### **시나리오 2: 이미 동의권 조항에 서명한 경우**

✅ **즉시 실행 사항**:
1. **주주간 계약서 별도 작성** → 창업 멤버 간 "VC 동의 거부 시 공동 대응" 조항 추가
2. **이사회 운영 규정 제정** → 동의권 발동 요건을 명확히 정의 (서면 양식, 기한 등)
3. **중대 의사결정 사전 소통** → 분기별 VC 미팅에서 계획 공유 (사후 거부 방지)

   - 무료 1차 상담 + 우대 수임료

3. **TIPS 프로그램 선정 시**
   - 투자 계약 전문가 멘토링 무료 제공

## [후속 질문 제안]

귀사의 성장 단계에서 추가로 고민하시면 좋을 사항:

1. **"주주간 계약(SHA)에 Tag-Along/Drag-Along 조항이 있나요?"**
   → Exit 전략과 직결되는 중요 조항입니다.

2. **"현재 대표이사 지분이 몇 %이고, 추가 희석 방어 전략은 있나요?"**
   → 시리즈 B, C 진행 시 경영권 유지 계획이 필요합니다.

3. **"투자금 집행 계획이 구체적으로 문서화되어 있나요?"**
   → 동의권 발동을 최소화하는 예산 운영이 핵심입니다.

### 💡 **최종 조언**

**"동의권은 투자자의 '안전벨트', 하지만 운전대는 대표님이 잡고 계셔야 합니다."**

- 협의권 중심으로 최대한 협상하되, 
- 수용 불가피한 동의권은 **"금액/비율 기준을 최대한 높게"** 설정하고,
- 반드시 **투자 전문 변호사의 최종 검토**를 받으십시오.

계약서는 한 번 서명하면 되돌리기 어렵습니다. 단 며칠의 검토 시간이 향후 5년의 경영 자율성을 결정합니다.
