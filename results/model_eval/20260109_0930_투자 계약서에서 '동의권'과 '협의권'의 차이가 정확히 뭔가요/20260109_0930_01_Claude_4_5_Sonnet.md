# Claude 4.5 Sonnet  응답 (1/3회차)

## 응답 내용
# 법률/계약 전문가로서 답변드리겠습니다

**[상황 진단]**
프리 시리즈 A 단계에서 VC가 강력한 통제권을 요구하는 전형적인 계약서 구조로 보입니다. 현재 개발팀 비중이 높은 기술 중심 조직으로, 빠른 의사결정이 필요한 시점에 '동의권' 조항이 과다하면 경영 병목이 발생할 수 있습니다.

---

## 📌 핵심 차이점 비교표

| 구분 | 동의권 (Consent Right) | 협의권 (Consultation Right) |
|------|----------------------|---------------------------|
| **법적 효력** | ✅ **거부권 존재** (동의 없으면 실행 불가) | ⚠️ **의견 청취 의무**만 존재 (최종 결정은 대표) |
| **의사결정 속도** | 느림 (VC 검토·승인 필수) | 빠름 (형식적 보고 후 진행 가능) |
| **경영 자율성** | 🔴 **낮음** | 🟢 **높음** |
| **투자자 보호** | 강력 | 약함 |
| **분쟁 소지** | 높음 (해석 다툼 여지) | 낮음 |

**현실적 위험도**: **동의권 > 협의권** (경영 마비 가능성 측면)

---

## ⚠️ 귀사의 구체적 리스크 3가지

### 1️⃣ **신속한 증자 불가능**
- **시나리오**: 반도체 업체 대형 계약 수주 → 긴급 개발 인력 7명 추가 필요 → 긴급 유상증자 시도
- **동의권 조항 시**: VC가 밸류에이션 재협상 요구 → 2~3주 지연 → **계약 유실**
- **협의권 조항 시**: 보고 후 즉시 진행 가능

### 2️⃣ **기술 제휴 지연**
- **예시**: 자동차 부품사와 공동 개발 계약 체결 시 "중요 계약"으로 해석되어 동의 필요
- **문제점**: VC가 재무 조건 검토하느라 14일 소요 → 경쟁사에 기회 선점됨

### 3️⃣ **자산 처분 제약**
- **예시**: 불필요한 검사 장비 매각으로 운영자금 확보
- **동의권 시**: VC가 "자산 가치 하락" 이유로 거부 → 자금난 심화

## ✅ 즉시 실행 가능한 협상 전략 (3단계)

### **STEP 1: 조항별 위험도 분류 (오늘~내일)**

```
[동의권 유지 가능 항목]
✓ 정관 변경
✓ 합병·분할·해산
✓ 대표이사 해임

[협의권으로 전환 요구 항목]
✓ 3억 원 이하 유상증자
✓ 일상적 영업 계약 (연 매출 10% 미만)
✓ 1억 원 이하 자산 매각
```

### **STEP 2: 협상 카드 마련 (이번 주)**

**VC에게 제시할 타협안**:
```
"동의권은 인정하되, 금액 기준선(Threshold) 설정을 요청합니다."

예시 문구:
- "5억 원 초과 거래에 한해 동의권 발동"
- "연간 유상증자 1회는 사전 승인 절차 면제"
- "긴급 경영 판단 시 이사회 결의로 갈음 가능"
```

### **STEP 3: 변호사 검토 필수 (다음 주)**

**준비 서류**:
1. 현재 투자 계약서 초안
2. 향후 6개월 경영 계획 (증자·채용·장비 투자 일정)
3. 유사 업종 표준 계약서 샘플 (K-Startup 제공 자료 활용)

**추천 법무법인**: 
- 스타트업 전문 변호사 (시간당 30~50만 원)
- 중소벤처기업진흥공단 **무료 법률 자문** (연 3회 제공)

## 🚨 리스크 체크리스트

- [ ] **Deadlock(교착상태) 조항** 확인: 동의권 행사 시 거부 사유 명시 의무가 있는가?
- [ ] **Deemed Consent(간주 동의)** 조항 삽입: "7일 내 답변 없으면 동의로 간주" 문구 추가
- [ ] **이사회 구성**: VC가 이사진 과반을 차지하는가? (최악의 경우 이사회 동의=VC 동의)
- [ ] **Put Option(매도청구권)**: 동의권 불이행 시 VC가 지분 환매 요구 가능한지 확인
