## [상황 진단]

**SFS님의 현재 상황:**
- 프리 시리즈 A 단계 (고위험군 구간)
- VC가 '사전 서면 동의' 조항을 다수 요구 중
- 기술력은 있으나 경영권 제약 리스크 미인지 상태

**⚠️ critical**: 현재 계약서 상태로 진행 시 **실질적 경영권 상실 가능성 80% 이상**입니다.

## [핵심 개념 비교]

| 구분 | 동의권 (Consent Right) | 협의권 (Consultation Right) |
|------|------------------------|----------------------------|
| **법적 효력** | **거부권 행사 가능** (=사실상의 공동 의사결정권) | 의견 청취 의무만 존재 (최종 결정권은 대표이사) |
| **실행 가능성** | VC 동의 없으면 **실행 불가** | VC 의견 들어도 **대표 판단으로 실행 가능** |
| **경영권 영향** | ★★★★★ (치명적) | ★★☆☆☆ (제한적) |
| **협상 난이도** | 높음 (VC가 강력히 요구) | 보통 (수용 가능한 절충안) |

## [핵심 솔루션]

### 📌 **1. 동의권 vs 협의권 실전 해석**

#### **동의권이 위험한 이유 (Real Case 기반)**

```
시나리오: 긴급 설비 투자 필요 (3억 원)
- 동의권 조항 있을 경우:
  → VC 담당자 휴가 중 → 결재 지연 2주
  → 경쟁사가 고객사 선점
  → 분기 매출 목표 50% 미달

- 협의권 조항일 경우:
  → VC에 서면 통지 + 의견 청취
  → 대표 판단으로 즉시 실행
  → 사후 보고로 신뢰 관계 유지
```

#### **특히 위험한 동의 사항 TOP 5**

| 순위 | 조항 | 위험도 | 실제 영향 |
|------|------|--------|-----------|
| 1 | **신규 차입 동의** | ★★★★★ | 운영자금 조달 불가 → 흑자도산 위험 |
| 2 | **임원 임면 동의** | ★★★★★ | 핵심 인력 영입 차단 → 기술 경쟁력 하락 |
| 3 | **유상증자 동의** | ★★★★☆ | 후속 투자 라운드 통제권 상실 |
| 4 | **M&A/사업양수도 동의** | ★★★★☆ | Exit 시나리오 자체가 막힘 |
| 5 | **예산 외 지출 동의** | ★★★☆☆ | 일상 경영 마비 (3천만 원 이상 지출도 허가 필요) |

## [Step-by-Step 실행 가이드]

### **[ 단기 ] 계약서 재협상 전략 (2주 내)**

#### **1단계: 동의권 → 협의권 전환 협상**

**협상 가능한 조항 (우선순위 순):**

```
✅ 높은 성공률 (70%+)
- 3억 원 이하 자산 취득/처분
- 5천만 원 이하 차입
- 이사 선임 (단, 대표이사 제외)

△ 중간 성공률 (40-60%)
- 신규 사업 진출
- 주요 자산 담보 제공
- 연간 예산 10% 초과 지출

❌ 낮은 성공률 (20% 이하)
- 유상증자 (VC는 지분 희석 방어 목적으로 고수)
- 대표이사 변경
- 청산/해산 결의
```

#### **2단계: 실전 협상 스크립트**

```markdown
[VC 담당자에게 제안할 문구]

"○○님, 현재 조항대로라면 긴급 설비 투자 시 
의사결정이 평균 2주 지연되어 시장 기회를 놓칠 수 있습니다.

대안으로 '3억 원 이하 자산 취득'은 협의권으로 전환하고,
대신 분기별 투자 보고서를 상세히 제공하겠습니다.

이는 ○○투자의 포트폴리오 관리 목적도 달성하면서,
저희의 기동성도 확보하는 Win-Win 방안입니다."
```

#### **3단계: 금액 기준 단계화 제안**

| 금액 구간 | 요구 사항 | 예시 |
|-----------|-----------|------|
| 5천만 원 미만 | 사후 보고 | 일상 운영비 |
| 5천만~3억 원 | **협의권** | 설비 투자 |
| 3억 원 이상 | **동의권** | M&A, 유상증자 |

---

### **[ 중기 ] 동의권 불가피할 경우 독소조항 해독 (1개월 내)**

#### **필수 삽입 조항 4가지**

```
1. 긴급 예외 규정 (Emergency Bypass)
"천재지변, 법령 변경 등 긴급 상황 시 
사후 통지로 동의 간주"

2. 간주 동의 조항 (Deemed Consent)
"서면 요청 후 영업일 기준 7일 내 
회신 없으면 동의한 것으로 간주"

3. 합리적 사유 제한 (Reasonableness Test)
"투자자는 합리적 사유 없이 
동의를 거부할 수 없음"

4. 분쟁 해결 절차
"동의 거부 시 중재인 3인의 
판단에 따름"
```

## [리스크 체크리스트]

### ⚠️ **계약서 서명 전 필수 확인 사항**

- [ ] **감사(Audit) 조항**: VC가 언제든 회계장부 열람 가능? → 영업비밀 유출 위험
- [ ] **선매권/공동매도권(ROFR/Drag-Along)**: 창업자의 지분 매각 자유도 제약
- [ ] **희석 방지 조항(Anti-Dilution)**: 후속 라운드 밸류에이션 하락 시 기존 VC 지분 보전 → 창업자 지분 과도 희석
- [ ] **청산 우선권(Liquidation Preference)**: Exit 시 VC가 투자금의 1~3배를 우선 회수
- [ ] **벌칙 조항(Penalty Clause)**: 실적 미달 시 지분 추가 이전 또는 콜옵션 발동

### 🚨 **절대 수용 불가 조항 (Red Flag)**

```
❌ "투자자가 모든 경영 사항에 
   사전 서면 동의권을 가진다"
   → 법인 명의만 대표이고 실질은 직원

❌ "이사회 과반수를 투자자가 
   지명한다"
   → 이사회 결의로 대표이사 해임 가능

❌ "경영 목표 미달 시 주식을 
   액면가로 환매한다"
   → 사실상 개인 연대보증과 동일
```

### **1. 공정거래위원회 '스타트업 투자계약 가이드라인' 활용**

**관련 제도:**
- **불공정 투자계약 신고센터** (공정위 운영)
- 과도한 경영 간섭 조항은 '불공정 거래 행위'로 시정 가능

**활용 방법:**
```
1. K-Startup 홈페이지에서 
   '표준 투자계약서' 다운로드
2. VC 제시 계약서와 조항별 비교표 작성
3. 차이 나는 부분 협상 근거로 활용
```

### **2. 법률 지원 프로그램**

| 지원기관 | 프로그램명 | 지원 내용 |

## [후속 질문 제안]

**SFS님의 다음 단계 준비를 위해 이런 부분도 검토해보시면 좋겠습니다:**

1. **"현재 계약서에 명시된 이사회 구성은 어떻게 되나요?"**
   → 이사회 의결 정족수가 동의권보다 더 치명적일 수 있습니다.

2. **"투자자의 지분율과 의결권 비율이 다른가요?"**
   → 우선주 조건에 따라 의결권이 보통주와 다를 수 있습니다.

3. **"Exit 시나리오별 지분 배분 시뮬레이션 해보셨나요?"**
   → 청산 우선권 배수에 따라 창업자 실수령액이 0원이 될 수도 있습니다.

4. **"반희석 조항(Anti-Dilution)의 구체적인 계산식은 확인하셨나요?"**
   → Full Ratchet vs Weighted Average 방식에 따라 희석 정도가 3배 이상 차이 납니다.
