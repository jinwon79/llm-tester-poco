### 파악된 핵심 이슈
글로벌 가전사와의 대규모 계약 체결은 성장의 신호이지만, **복잡한 반품 및 할인 조건**은 다음과 같은 재무 리스크를 내포하고 있습니다:

1. **수익인식 시점의 불확실성** (K-IFRS 1115 수익인식 기준)
2. **매출채권 회수 리스크 및 현금흐름 악화 가능성**
3. **반품충당부채 및 판매할인 추정 오류로 인한 실적 왜곡**
4. **외부감사 시 회계처리 적정성 검증 대응 미흡**

### 추가 파악 필요 정보

- [ ] 할인 구조 (물량할인, 판촉할인, 리베이트 등)
- [ ] 현재 회계처리 방식 (수익인식 시점, 충당부채 설정 여부)
- [ ] 외부감사 대상 여부 및 감사법인 규모

## 💡 [핵심 솔루션]

### 1️⃣ 수익인식 기준 재정립 (K-IFRS 1115 준수)

| 단계 | 회계처리 | 실무 적용 |
|------|----------|-----------|
| **1단계: 계약 식별** | 각 계약별 조건 문서화 | 계약서 내 반품/할인 조항 별도 관리 |
| **2단계: 수행의무 구분** | 제품 공급 vs. 추가 서비스 분리 | 물류, A/S 등 별도 매출 구분 |
| **3단계: 거래가격 산정** | **변동대가 추정** (기댓값법) | 과거 반품률 데이터 기반 산출 |
| **4단계: 거래가격 배분** | 개별 수행의무별 상대적 가치 | 제품별 표준원가 기준 배분 |
| **5단계: 수익 인식** | 통제 이전 시점 인식 | 출고 vs. 검수완료 기준 명확화 |

### 2️⃣ 반품충당부채 설정 (Required)

```
[회계처리 예시]
매출 발생 시:
  (차) 매출채권 1,000  (대) 매출 900
                           반품충당부채 100

실제 반품 시:
  (차) 반품충당부채 100  (대) 매출채권 100
```

**설정 기준**:
- 과거 3년간 실제 반품률 데이터 분석
- 계약서상 반품 가능 기간 및 조건 반영
- 분기별 충당부채 적정성 검토 (Over/Under 조정)

### 3️⃣ 판매할인 관리 체계 구축

| 할인 유형 | 회계처리 | 관리 포인트 |
|-----------|----------|-------------|
| **즉시 할인** | 순액 인식 (매출 차감) | 계약서 명기 필수 |
| **물량할인** | 추정 후 충당금 설정 | 분기별 실적 기준 재계산 |
| **연간 리베이트** | 월별 추정 누적 | 계약 조건 변경 시 즉시 반영 |

## 🚀 [Step-by-Step 실행 가이드]

### 📅 **1주차: 긴급 점검**
1. **계약서 전수 검토**
   - 반품 조건(%, 기간, 품질 기준) 항목별 체크리스트 작성
   - 할인 조건(트리거, 지급시기, 계산방식) 별도 정리
   
2. **현행 회계처리 Gap 분석**
   - 현재 수익인식 시점 vs. K-IFRS 기준 비교표 작성
   - 반품충당부채 미설정 시 → 즉시 회계법인 자문 의뢰

### 📅 **2~4주차: 시스템 구축**
3. **ERP 수익인식 모듈 개선**
   - 계약별 변동대가 자동 계산 로직 구현
   - 반품률 실시간 추적 대시보드 구축
   
4. **내부 프로세스 표준화**
   ```
   [수익인식 체크리스트]
   □ 계약서 검토 (법무팀)
   □ 변동대가 산정 (재무팀)
   □ 수익인식 승인 (CFO)
   □ 분기별 충당부채 재검토
   ```

### 📅 **1~3개월: 외부 검증**
5. **감사법인 사전 협의**
   - 중간감사(반기) 시 회계처리 적정성 검증
   - 필요 시 회계정책 변경 공시 준비

6. **세무 리스크 점검** *(세무 전문가 협업)*
   - 법인세 신고 시 익금산입/손금산입 검토
   - 부가가치세 대손세액공제 요건 확인

## ⚠️ [리스크 체크리스트]

### 🔴 Critical (즉시 조치)

- [ ] 계약서에 반품 조건 명시 없이 구두 합의 → 법적 분쟁 가능성
- [ ] 변동대가 추정 근거 부족 → 회계감리 지적 사항

### 🟡 Important (1개월 내)
- [ ] 할인 조건 변경 시 과거 계약에 소급 적용 여부 불명확
- [ ] 외화 거래 시 환율 변동에 따른 변동대가 재계산 미비
- [ ] 재무제표 주석 공시 부족 (수익인식 정책 상세 설명)

### 🟢 Preventive (분기별 점검)
- [ ] 충당부채 추정 오류 누적 시 일시적 실적 급변동
- [ ] 물류팀과 재무팀 간 재고 데이터 불일치
- [ ] 고객사 재무 악화 시 대손충당금 추가 설정 필요
