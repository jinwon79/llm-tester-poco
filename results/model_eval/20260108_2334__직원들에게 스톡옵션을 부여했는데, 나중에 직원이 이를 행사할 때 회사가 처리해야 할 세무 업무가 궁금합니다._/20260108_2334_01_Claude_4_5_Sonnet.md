## 🔍 **[상황 진단]**

귀사는 **비상장 핀테크 스타트업**으로서 핵심 인력 5명에게 부여한 스톡옵션의 행사 시점이 도래한 상황입니다. 

**추가 확인이 필요한 정보:**
- 스톡옵션 부여 시 행사가격은 어떻게 설정되었나요?
- 현재 주식의 1주당 시가는 얼마인가요? (최근 투자 유치 시 주가 기준)
- 세무서에 '주식매수선택권 부여 명세서' 제출 여부
- 스톡옵션 유형: 일반 스톡옵션 vs 세제적격 스톡옵션

**현재 정보 기준으로 핵심 세무 절차를 안내드리겠습니다.**

## 💼 **[핵심 세무 처리 사항]**

### **1. 행사 시점의 세무 구조**

| 구분 | 일반 스톡옵션 | 세제적격 스톡옵션 |
|------|--------------|------------------|
| **행사 시점 과세** | 근로소득세 과세 (최고 45%) | **과세 이연** (행사 시 비과세) |
| **과세 기준** | (시가 - 행사가격) × 행사 주식 수 | 양도 시점까지 과세 없음 |
| **회사 원천징수 의무** | ✅ 있음 | ❌ 없음 (양도 시 개인이 신고) |
| **적용 요건** | 제한 없음 | 벤처기업, 중소·중견기업만 가능 |

---

### **2. 회사가 즉시 처리해야 할 세무 업무 체크리스트**

#### ✅ **STEP 1: 행사 시점 (D-Day)**
```
1. 주식 시가 평가
   - 최근 유상증자/투자 유치 시 주가 적용
   - 없을 경우: 순자산가치, 수익가치 가중평균 (세법 기준)
   
2. 과세표준 계산
   과세대상 = (시가 - 행사가격) × 행사 주식 수
   
3. 원천징수 세액 산출
   - 근로소득세율 적용 (6%~45% 누진세율)
   - 지방소득세 10% 별도
```

#### ✅ **STEP 2: 익월 10일까지 (원천징수 신고·납부)**
```
국세청 홈택스 접속
→ '근로소득 간이지급명세서(반기)' 제출
→ 원천징수이행상황신고서 제출
→ 세액 납부 (가상계좌 또는 전자납부)
```

#### ✅ **STEP 3: 다음 해 2월 말 (연말정산)**
```
- 근로소득 지급명세서에 스톡옵션 행사 이익 포함
- 직원 개인의 근로소득 합산하여 최종 정산
```

## 📋 **[단계별 실행 가이드]**

### **🟢 즉시 실행 (이번 주 내)**

1. **세무사 선임 또는 상담**
   - 비상장 주식 시가 평가는 복잡하므로 전문가 도움 필수
   - 추천: 스타트업 전문 세무법인 (예: 세무법인 가감승제, 더존 등)

2. **행사 신청자별 과세액 시뮬레이션**
   ```
   [예시 계산]
   - 행사가격: 10,000원
   - 현재 시가: 50,000원
   - 행사 수량: 1,000주
   
   과세표준 = (50,000 - 10,000) × 1,000 = 4,000만 원
   소득세 = 4,000만 × 24% - 522만 = 438만 원
   지방소득세 = 43.8만 원
   → 총 원천징수액: 약 482만 원
   ```

3. **직원 사전 안내 및 동의**
   - "행사 시 세금이 XX만 원 발생하며, 회사가 원천징수합니다"
   - 현금 납부 곤란 시: '주식 일부 매각 후 행사' 또는 '회사 대여' 협의

---

### **🟡 1개월 내 정비**

4. **세제적격 스톡옵션 전환 검토**
   - 귀사가 **벤처기업 인증**을 보유한 경우, 향후 부여 분은 세제적격으로 전환 권장
   - 효과: 직원 세 부담 최소화 + 회사 원천징수 부담 제로

5. **스톡옵션 관리 시스템 도입**
   - 추천 솔루션: 벤처스퀘어, 캡테이블(CapTable) 관리 플랫폼
   - 기능: 행사 일정 알림, 세액 자동 계산, 서류 자동 생성

---

### **🔵 3개월 내 구축 (재발 방지)**

6. **스톡옵션 행사 규정 문서화**
   - 회사 내부 규정에 '행사 절차, 세무 처리, 대금 결제 방식' 명문화
   - 표준 서식: 행사신청서, 원천징수 동의서

7. **IR 자료에 희석 효과 반영**
   - 향후 투자 유치 시, 스톡옵션 행사로 인한 지분 희석률 사전 공시
