## 🔍 **2. 법률적 핵심 쟁점 분석**

### (1) 갱신기대권(갱신거절의 제한)이란?
- **정의**: 기간제 근로자가 반복 갱신 또는 사용자의 언행으로 인해 '계약이 계속될 것'이라는 **정당한 기대**를 갖게 된 경우, 사용자가 **정당한 이유 없이 계약을 종료**하면 부당해고로 보는 법리
- **근거**: 기간제법 제4조 + 대법원 판례(2018다264857 등)

### (2) 귀사 상황의 위험 요소

| 리스크 요인 | 귀사 적용 상황 | 법원 인정 가능성 |
|------------|--------------|----------------|
| **반복 갱신 여부** | 2년(1회 갱신 시 고위험) | ⚠️ 중~高 |
| **구두 약속** | "성과 좋으면 연장" 발언 | ⚠️ 高 (증거 존재 시) |
| **동종 직군 관행** | 다른 강사들 통상 갱신 시 | ⚠️ 高 |
| **갑작스런 번복** | 경영난 사유의 객관성 부족 시 | ⚠️ 高 |

### (3) 판례 기준 (대법원 2018다264857)
✅ 아래 **3가지 이상** 해당 시 갱신기대권 인정 가능성 매우 높음:

1. 계약이 **2회 이상 갱신**됨
2. 사용자가 계약 갱신에 대해 **긍정적 발언** (구두 포함)
3. **동종 직군의 통상적 갱신 관행** 존재
4. 업무의 **상시·지속성** (교육 강사는 해당)
5. 근로자의 **귀책사유 없음**

## 🛡️ **4. 즉시 실행 가능한 리스크 최소화 전략**

### 📌 **STEP 1: 긴급 증거 확보 (1주 내)**

#### ✅ 해야 할 일
```
1. 경영난 객관적 증빙 확보
   - 최근 3개월 손익계산서
   - 수강생 감소 추이 데이터
   - 이사회 의사록 (구조조정 결정 과정)

2. 계약서 및 인사 기록 점검
   - 해당 강사의 계약서 원본 확인
   - 과거 갱신 시 조건 및 사유 문서화
   - 이메일/메신저 내 '갱신 암시' 발언 검색

3. 동종 직군 관행 정리
   - 다른 강사 15명의 계약 갱신 이력 표 작성
```

#### ❌ 절대 금지 사항

- 해당 강사 앞에서 다른 강사의 계약연장 언급

### 📌 **STEP 2: 법률 검토 및 대응 시나리오 수립 (2주 내)**

#### 시나리오 A: **합의 종료 (권장)**
```
[목표] 분쟁 사전 차단 + 비용 최소화

1. 노무사/변호사와 합의서 초안 작성
2. 제안 패키지 설계
   - 위로금: 3개월분 급여 (약 900만원)
   - 퇴직금 조기 지급
   - 재취업 추천서 제공
   - 향후 프리랜서 강의 기회 우선 부여

3. 면담 시나리오 작성
   - 감정적 대응 방지 스크립트
   - "귀하의 기여를 인정하나, 불가피한 경영상 이유" 프레임
```

**→ 예상 비용: 1,200만원 (소송 대비 85% 절감)**

#### 시나리오 B: **계약 종료 강행 (고위험)**
```
[전제 조건] 아래 모두 충족 시에만 시도

✅ 경영난이 객관적으로 입증 가능 (매출 30% 이상 급감 등)
✅ 구두 약속이 녹취/문서로 남지 않음
✅ 다른 강사들도 동시에 계약 종료 (선별적 종료 아님)
✅ 해당 강사가 소송 제기 시 전액 패소 감수 가능

→ 이 경우에도 '계약 종료 사유 서면 통지서' 작성 필수
```
