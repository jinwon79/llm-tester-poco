## 응답 내용
# 노무/인사 분야 전문가로서 답변드리겠습니다

귀사의 상황은 **'기대권(갱신기대권) 인정 여부'**와 **'계약 해지 절차의 적법성'**이 핵심 쟁점입니다. 구두 소통 내역과 경영난이라는 불가피한 사유가 충돌하는 전형적인 노무 리스크 사안입니다.

## 📊 상황 진단

### 파악된 정보
- 계약 기간: 2년 (만료 임박)
- 계약 형태: 전문 계약직 (기간제 근로자)
- 과거 소통: "계약 연장 가능성" 구두 언급
- 해지 사유: 경영난 (갑작스러운 변화)
- 성과: 우수 (이는 오히려 갱신기대권 인정에 불리하게 작용할 수 있음)

### 추가 확인 필요 사항
더 정확한 법적 판단을 위해 다음 정보를 확인해주시면 정밀 진단이 가능합니다:

1. **과거 계약 갱신 이력**: 해당 직원이 2년 전 첫 계약인가요, 아니면 이미 1회 이상 갱신된 상태인가요?
2. **구두 소통의 구체성**: "연장 검토 중" vs "확정적으로 연장하겠다"의 차이
3. **동종 직군 타 직원**: 같은 조건의 다른 강사들은 계약이 갱신되었나요?
4. **경영난 입증 자료**: 매출 감소, 사업 축소 등 객관적 증빙 가능 여부
5. **계약서 상 갱신 조항**: "자동 갱신", "협의 후 갱신" 등의 문구 유무

## 🎯 핵심 솔루션: 갱신기대권 법리 분석

### 1️⃣ 갱신기대권이란?
기간제 근로자가 **"합리적으로 계약 갱신을 기대할 수 있는 상황"**이 형성된 경우, 사용자가 정당한 이유 없이 갱신을 거절하면 **부당해고(실질적 해고)**로 간주될 수 있는 법리입니다.

### 2️⃣ 판례 기준 (대법원 2020다248809 등)
다음 요소를 **종합적으로** 고려합니다:

| 판단 요소 | 귀사 사례 적용 |
|---------|-------------|
| **계약 갱신 횟수** | 2년 첫 계약 → 기대권 ↓ / 이미 1회 갱신 → 기대권 ↑ |
| **근속 기간** | 2년은 중립적 (3년 이상 시 기대권 ↑) |
| **사용자의 언행** | ⚠️ **구두로 "연장 가능성" 언급** → 기대권 형성 가능성 ↑ |
| **동종 근로자 처우** | 타 강사 갱신 시 차별 문제 발생 가능 |
| **업무 성과** | ⚠️ **우수한 성과** → 갱신 거절의 합리성 ↓ |
| **경영상 필요성** | 객관적 경영난 입증 시 정당한 사유 인정 가능 |

### 🚨 귀사의 리스크 수준: **중상(中上)**
- **위험 요인**: 구두 약속 + 우수 성과 + (추정) 과거 갱신 이력
- **방어 가능성**: 경영난이라는 불가피한 사유 존재

## 🛠️ Step-by-Step 실행 가이드

### 📌 즉시 실행 (D-30일 전)
#### ✅ 1단계: 경영난 입증 자료 준비
```
필수 서류:
□ 최근 3개월 손익계산서
□ 수강생 감소 추이 데이터
□ 강사진 재배치 계획서
□ 이사회 의사록 (인건비 절감 결정 등)
```

#### ✅ 2단계: 계약서 재검토
- 갱신 관련 조항 확인 ("쌍방 합의 시 갱신 가능" vs "자동 갱신" 구분)
- 갱신 거절 사유 명시 여부 확인

#### ✅ 3단계: 소통 기록 정리
- 구두 약속의 **맥락** 재구성 ("확정" vs "검토 중"의 뉘앙스)
- 이메일, 문자, 회의록 등 문서화된 증거 취합

### 📌 계약 종료 통보 전 (D-15일 전)
#### ✅ 4단계: 사전 면담 실시
```
면담 스크립트 예시:
"○○님, 그동안 우수한 성과에 감사드립니다. 
다만 코로나 여파로 인한 수강생 급감(구체적 수치 제시)으로 
불가피하게 강사진 재편성을 진행 중입니다. 
계약 만료 시점에 갱신이 어려울 것 같아 미리 말씀드립니다."
```
- ⚠️ **"성과가 나빠서"라는 표현 절대 금지** (갱신기대권 역으로 강화됨)

#### ✅ 5단계: 대안 제시 (리스크 완화 전략)
- 프리랜서 강사로 전환 후 건당 수업료 지급 제안
- 타 교육기관 추천서 작성 지원
- 퇴직 위로금 지급 (법적 의무는 아니나 분쟁 예방 효과)

### 📌 통보 시점 (D-Day)
#### ✅ 6단계: 서면 통보 (내용증명)
```
필수 기재사항:
1. 계약 만료일 명시
2. 갱신하지 않는 사유 (경영상 이유 구체적 서술)
3. 법정 예고 기간 준수 (기간제법 제4조 - 만료 30일 전 통보 권장)
4. 퇴직금 등 정산 일정
```

## 📋 추가 법률 검토 사항

- 동일 직무 정규직 대비 불합리한 차별이 있었다면 별도 소송 리스크 존재

### 2️⃣ 근로기준법 제23조 (부당해고 금지)
- 갱신기대권이 인정되면 기간 만료가 아닌 '해고'로 재분류
- 정당한 이유 + 적법 절차 모두 입증 필요

### 3️⃣ 노동위원회 구제신청
- 직원이 부당해고 구제 신청 시 **원직 복직 명령** 가능성
- 구제명령 확정 시 중간 임금(임금 손실분) 지급 의무
