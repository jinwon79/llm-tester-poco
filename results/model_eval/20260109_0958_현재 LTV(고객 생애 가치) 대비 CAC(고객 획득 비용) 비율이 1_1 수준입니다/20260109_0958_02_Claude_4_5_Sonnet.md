## 🔍 1. 상황 진단 (Critical Analysis)

### 현재 구조적 문제점
```
문제의 본질: "고객 유입 ≠ 고객 정착"
├─ CAC 과다: 유튜브 광고 의존 → 성과형 마케팅 효율 저하
├─ LTV 저조: 1개월 이탈 → 리텐션(Retention) 구조 붕괴
└─ 단위경제성 악화: 매출 < 마케팅비용 → 투자 유치 불가능
```

### 더 정확한 컨설팅을 위한 추가 정보 요청
- **월평균 CAC**: 구체적 금액 (예: 50,000원?)
- **월 구독료**: 가격 정책 (예: 29,000원?)
- **1개월 이탈률**: % (예: 70%?)
- **활성 사용자 중 3개월 이상 유지율**: % (투자자가 중시하는 핵심 지표)

> 💡 **현재 정보 기준으로 즉각 실행 가능한 솔루션을 제시하되, 위 수치 확보 시 정밀 전략 수립 가능합니다.**

## 🛠️ 3. Step-by-Step 실행 가이드

### 🔴 Phase 1: 긴급 리텐션 개선 (1~2개월, LTV 150% 목표)

**문제**: 1개월 이탈 = "습관 형성 실패"

#### Action 1-1: 온보딩 강화 ("첫 7일이 전부다")
```
✅ 1주차 학습 커리큘럼 강제 설계
   - Day 1: AI 튜터 첫 대화 (5분 초단위 미션)
   - Day 3: 첫 화상 수업 예약 (예약률 70% 목표)
   - Day 7: "성취 리포트" 자동 발송 (학습 시간 가시화)

✅ 푸시 알림 시나리오 설계
   - 하루 놓침 → "오늘의 5분 문장" 자동 전송
   - 3일 놓침 → 마케팅팀 → 1:1 카톡 상담
```

#### Action 1-2: 구독 해지 방어 시스템
```
해지 신청 시 팝업:
"1개월 더 할인 쿠폰 (50% OFF)" 제공
→ 2개월 이상 유지 시 LTV 2배 달성
```

#### Action 1-3: 코호트 분석 즉시 착수
```python
# 개발팀 태스크 (2주 소요)
- 가입일 기준 1/2/3개월 리텐션률 대시보드 구축
- 이탈 예측 모델 (ML 불필요, 단순 행동 패턴 분석)
  예: "7일 내 AI 튜터 3회 미만 사용 → 80% 이탈"
```

---

### 🟡 Phase 2: CAC 절감 (2~3개월, 30% 목표)

**문제**: 유튜브 광고 = 비효율적 퍼널 (관심 → 구매 전환율 낮음)

#### Action 2-1: 퍼포먼스 마케팅 전환
```
✅ 메타(페이스북/인스타) 광고 테스트
   - 타겟: "토익, 오픽" 키워드 관심사 + 25~35세
   - 크리에이티브: Before/After 짧은 후기 영상 (15초)
   - 예산 배분: 유튜브 30% → 메타 40% + 구글 30%

✅ 검색 광고 강화
   - 네이버 파워링크: "화상 영어 추천" 등 롱테일 키워드
   - 구글 애즈: "영어 회화 앱 무료 체험"
```

#### Action 2-2: 오가닉 채널 육성 (CAC=0 목표)
```
- 블로그 SEO: "직장인 영어 공부법" 등 월 20개 콘텐츠
- 유튜브 쇼츠: AI 튜터 사용 팁 (주 3회, 바이럴 노림)
- 인플루언서 협업: 마이크로 인플루언서 10명 (제품 제공)
```

---

### 🟢 Phase 3: 수익 구조 다각화 (3개월~, LTV 추가 30% 상승)

#### Action 3-1: 프리미엄 요금제 신설
```
현재: 단일 요금제 (예: 월 29,000원)
추가:
  - 프리미엄 (49,000원): 1:1 원어민 수업 무제한
  - 기업 B2B (팀 단위): 임직원 교육 패키지
```

#### Action 3-2: 연간 구독 유도
```
- 월 29,000원 vs. 연 290,000원 (2개월 무료)
→ 선결제 → 즉시 LTV 10배 상승 효과
```
