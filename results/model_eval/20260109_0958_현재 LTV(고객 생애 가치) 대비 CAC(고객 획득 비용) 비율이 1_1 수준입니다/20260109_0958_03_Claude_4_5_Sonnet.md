## 🔍 상황 진단

### 현재 구조 분석
```
[핵심 문제점]
✗ 고비용 유입 채널 (유튜브 광고) 의존
✗ 1개월 이탈률 과다 → LTV 저하의 직접적 원인
✗ 프로덕트-마켓 핏(PMF) 불완전 상태

[긍정 요소]
✓ 유입 자체는 성공적 (퍼널 상단 확보)
✓ AI+실시간 결합 모델 (차별화 요소 보유)
✓ 명확한 타겟층 (2030 직장인)
```

### 📌 더 정확한 컨설팅을 위해 필요한 정보
1. **현재 CAC/LTV 구체적 수치** (예: CAC 5만원, LTV 5만원?)
2. **월 구독료 및 평균 구독 기간** (예: 월 29,000원 × 평균 1.7개월?)
3. **1개월 유지율(Retention Rate)** (예: 30%만 2개월 이상?)
4. **코호트별 이탈 시점** (가입 직후 vs 7일 vs 30일?)
5. **경쟁사 대비 가격 경쟁력** (튜터링 vs 립 vs 스픽 등)

**아래는 일반적인 B2C 구독 모델 기준으로 전략을 제시하되, 위 정보 공유 시 맞춤 전략 제공 가능합니다.**

## 💡 핵심 솔루션: 2단계 동시 개선 전략

### 📈 [전략 A] LTV 2배 상승 → 5만원에서 10만원으로 (우선순위 ★★★)

| 실행 항목 | 구체적 방법 | 목표 지표 |
|---------|-----------|---------|
| **온보딩 최적화** | • 가입 후 72시간 집중 관리<br>• AI 튜터 첫 세션 무료 체험권 자동 발급<br>• 개인 맞춤 학습 플랜 즉시 제공 | 7일 이탈률 50% → 30% |
| **리텐션 트리거 설계** | • 7일차: 실시간 튜터 1:1 무료 세션 제공<br>• 14일차: 학습 성과 리포트 발송<br>• 21일차: 커뮤니티 챌린지 초대 | 30일 유지율 30% → 50% |
| **가격 구조 재설계** | • 월 단위 → 3개월/6개월 선결제 할인 유도<br>• "3개월 플랜 20% 할인" 프로모션 | 평균 계약 기간 1.7개월 → 3.5개월 |
| **프리미엄 업셀** | • 실시간 튜터 횟수 추가 패키지<br>• 기업 영어(비즈니스 전문) 별도 상품 | ARPU 30% 증가 |

**즉시 실행 가능한 액션 (이번 주 내)**
1. 가입 후 자동 발송 이메일 시퀀스 3개 제작 (웰컴-학습법 가이드-성공 사례)
2. 구글 시트로 코호트 분석표 작성 (가입월별 유지율 추적)
3. 기존 이탈 고객 100명 대상 간단 설문 (이탈 사유 1순위 파악)

---

### 💰 [전략 B] CAC 30% 절감 → 5만원에서 3.5만원으로 (우선순위 ★★)

| 실행 항목 | 구체적 방법 | 목표 지표 |
|---------|-----------|---------|
| **유튜브 광고 최적화** | • 전환율 낮은 크리에이티브 과감히 정리<br>• CPA 기준 상위 20% 소재만 집중 투자<br>• 숏폼(60초 이하) 비중 80%로 확대 | CPA 30% 개선 |
| **오가닉 채널 개발** | • 직장인 커뮤니티 (블라인드, 인스타 릴스) 공략<br>• "출근길 5분 영어" 콘텐츠 시리즈화<br>• 추천인 제도 (친구 초대 시 양쪽 1개월 무료) | 오가닉 유입 20% 확보 |
| **재방문 유도 (CRM)** | • 이탈 고객 재활성화 캠페인<br>• "복귀 웰컴백 50% 할인" | 재구매율 10% 확보 |

**이번 달 내 실행 항목**
1. 메타(인스타/페이스북) 광고 A/B 테스트 시작 (유튜브 의존도 낮추기)
2. 기존 고객 중 만족도 높은 50명에게 추천 코드 발급
3. 블로그/유튜브 '영어 회화 실수담' 콘텐츠 주 2회 발행

## 🎯 Step-by-Step 실행 가이드 (90일 로드맵)

### 1단계: 출혈 멈추기 (D+0~30일)
```
[Week 1-2] 긴급 리텐션 대책
□ 1개월 이탈 고객 전수 인터뷰 (최소 30명)
□ 온보딩 플로우 개편안 설계
□ 3개월 할인 플랜 런칭

[Week 3-4] 광고 효율화
□ 유튜브 광고 성과 하위 50% 소재 중단
□ 메타 광고 테스트 예산 30만원 배정
□ 추천인 제도 베타 오픈
```

### 2단계: 성장 엔진 구축 (D+31~60일)
```
□ 코호트 분석 기반 리텐션 전략 고도화
□ 오가닉 콘텐츠 50개 제작 (릴스/블로그)
□ 프리미엄 상품 테스트 론칭 (50명 대상)
```

### 3단계: 지표 검증 및 투자 준비 (D+61~90일)
```
□ LTV/CAC 비율 재측정 (목표: 2.5:1 이상)
□ 유닛 이코노믹스 정리 (투자 IR 자료화)
□ 시리즈 A 투자사 접촉 시작
```
