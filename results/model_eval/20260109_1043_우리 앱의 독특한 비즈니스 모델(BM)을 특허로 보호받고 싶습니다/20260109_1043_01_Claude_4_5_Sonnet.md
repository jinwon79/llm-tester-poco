# [법률(IP) + 경영전략 전문가 답변]

안녕하세요, **(주)핀포인트** 대표님.  
**지식재산권(IP) 전문 변호사 관점에서 비즈니스 모델 특허 가능성을 검토**하되, **경영전략 컨설턴트로서 실제 진입장벽 구축 방안**까지 통합 제안드리겠습니다.

## 📌 **상황 진단**

### ✅ 파악된 정보
- **사업 모델**: 주차 공간 실시간 경매(Auction) 매칭 플랫폼
- **핵심 차별점**: '유휴 공간 + 시간대별 동적 가격결정 + 옥션 알고리즘'
- **보호 목적**: 경쟁 진입 방지 및 초기 시장 선점

### ⚠️ 추가 확인 필요 사항
더 정확한 IP 전략 수립을 위해 아래 정보를 공유해주시면 맞춤 전략이 가능합니다:
1. **기술적 구현 방식**: 
   - 경매 알고리즘이 AI/머신러닝 기반인가요? (수요 예측, 동적 가격 조정)
   - 실시간 매칭에 독자적 기술(예: 위치 기반 최적화 로직)이 있나요?
2. **현재 개발 단계**: MVP 완성 여부 / 실제 거래 데이터 확보 여부
3. **경쟁사 조사**: 유사 특허(국내외) 검색 완료 여부

## 🔍 **핵심 솔루션: BM 특허 가능성과 전략**

### 1️⃣ **비즈니스 모델 특허, 가능한가?**

| 구분 | 내용 |
|------|------|
| **결론** | ✅ **"기술적 수단"과 결합 시 특허 가능** (단순 아이디어는 불가) |
| **한국 특허법** | **"자연법칙을 이용한 기술적 사상의 창작"**이어야 함<br>→ 소프트웨어 알고리즘, 시스템 구성도, 처리 흐름이 **기술적으로 구체화**되어야 함 |
| **판례 기준** | • 추상적 아이디어(예: "경매로 주차 공간 연결")만으로는 거절<br>• **컴퓨터 시스템과의 결합**으로 "기술적 효과" 입증 시 인정<br>예시: 특허법원 2018허5522 – 앱 기반 거래 알고리즘 특허 인정 |

---

### 2️⃣ **핀포인트 BM의 특허 가능 요소 분석**

#### ✅ **특허 가능성 높은 요소**
```
[청구항 1] 주차 공간 실시간 경매 매칭 시스템
- 서버가 건물주 단말로부터 "시간대별 주차 공간 정보"를 수신
- AI 기반 수요 예측 모델로 "시작 경매가" 자동 산정
- 운전자 단말의 입찰 데이터를 실시간 처리하여 "최적 매칭" 수행
- 결제 및 주차권 자동 발급까지 일괄 처리

→ 이것이 "단순 중개"가 아닌 **"기술적 처리 과정"**임을 입증
```

#### ⚠️ **특허 받기 어려운 요소**
- ❌ "주차 공간을 경매한다"는 추상적 아이디어 자체
- ❌ 단순 화면 UI/UX (디자인특허 별도 검토 필요)
- ❌ 비즈니스 규칙만 나열 (예: "입찰 최고가 낙찰")

## 📋 **Step-by-Step 실행 가이드**

### **[1단계] 특허 전 필수 작업 (착수 즉시~1개월)**

| 순서 | 액션 | 담당 | 비용 |
|------|------|------|------|
| 1 | **선행기술 조사**(Prior Art Search)<br>→ 키프리스(KIPRIS), 구글 특허 검색<br>→ "parking auction", "shared parking" 키워드 | 특허사무소 | 50~100만원 |
| 2 | **기술 명세서 작성**<br>→ 개발자와 협업: 알고리즘 플로우차트, 시스템 구성도 상세화<br>→ "입력→처리→출력" 각 단계의 기술적 특징 문서화 | 내부 + 변리사 | - |
| 3 | **발명자 회의**<br>→ 핵심 기술 3가지 이상 도출 (예: 동적 가격 알고리즘, 실시간 매칭 로직, 사기 방지 검증 시스템) | 대표+개발팀 | - |

```yaml
  - 독립항: "주차 공간 경매 매칭 시스템 및 그 방법"
  - 종속항: AI 가격 산정, 실시간 위치 기반 필터링, 결제 연동 등

단점: 심사 기간 1~2년 / 비용 높음 (300~500만원)
```

```yaml
3. 상표 출원: "핀포인트", "주차옥션" 등 브랜드 선점
```

## ⚠️ **리스크 체크리스트**

### 🔴 **특허 출원 시 주의사항**

1. **공지 예외 주장 불가**  
   - 이미 앱스토어 출시 or 투자 IR 자료 배포 시 "신규성 상실" 위험  
   → **즉시 출원** 후 1년 내 정식 출원 (우선권 주장)

2. **직무발명 분쟁 방지**  
   - 개발자와 **"직무발명보상규정"** 사전 합의 필수  
   → 특허 등록 시 보상금 지급 기준 명시 (예: 매출의 0.5%)

3. **해외 출원 고려**  
   - PCT 국제출원(비용 1,000만원~): 미국/중국 진출 시 필수  
   → 초기에는 **한국 우선 출원 후 판단** 권장

   → 법률 전문가: 침해 소송 vs 크로스 라이센싱 전략 검토

   → 법률 전문가: 배타적 거래 조항의 공정거래법 리스크 분석

   → 법률 전문가: 주차장 중개업 등록 요건 및 데이터 보관 의무 검토
