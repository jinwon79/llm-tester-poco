## 💡 핵심 결론

### **결론: 조건부 계속 가능 (단, 요건 충족 필수)**

창업세액감면(조세특례제한법 제6조)은 **본점 소재지 이전 자체로는 중단되지 않습니다**. 
다만, 다음 요건을 모두 충족해야 합니다:

| 구분 | 요건 | 귀사 적용 시 체크포인트 |
|------|------|------------------------|
| **창업일 기준** | 감면 개시일로부터 5년간 적용 | 창업 후 경과연도 확인 필요 |
| **수도권 제외 업종** | 지식기반산업 등 예외 업종 해당 여부 | ✅ SW개발업(C72201) 추정 시 **가능** |
| **중소기업 요건** | 상시근로자 평균 5인 이상 유지 | ✅ 현재 6명으로 충족 |
| **사업의 동일성** | 본질적 사업 내용 유지 | ✅ 전기차 알고리즘 사업 지속 |

## ⚠️ 핵심 리스크 체크리스트

### 1️⃣ **수도권 창업세액감면 제한**
```
[법령] 조세특례제한법 제6조 제3항
- 수도권 내 창업 시 '지식기반산업' 등 한정 업종만 감면 가능
```

**귀사의 업종 판단이 관건입니다:**

| 업종코드 | 감면 가능 여부 | 판단 기준 |
|---------|--------------|----------|
| **C72201** (소프트웨어 개발) | ✅ 가능 | 지식기반산업 해당 |
| **M71900** (기타 엔지니어링) | ⚠️ 검토 필요 | 주된 사업 실체 확인 |
| **G46900** (도소매업) | ❌ 불가 | 충전소 직접 운영 시 위험 |

**→ 사업자등록증의 '업태·종목'을 반드시 확인하세요.**

### 2️⃣ **이전 시점 타이밍 리스크**
- 사업연도 중 이전 시: 안분 계산 발생 가능
- 과세 당국 조사 시 '실질적 사업장' 판단 근거 제시 필요

## 📝 Step-by-Step 실행 가이드

### 🔹 **1단계: 즉시 실행 (이전 전 필수 확인)**

```
✓ [D-Day] 관할 세무서 사전 질의
  - 청주세무서 → 성동세무서 사전 통보
  - 업종코드 적격성 확인 (신청서식: 일반상담 신청서)
  
✓ [D-3일] 창업세액감면 명세서(세액감면신청서) 재검토
  - 최초 신청 시 기재한 사업장 소재지 vs. 현재 이전 계획
  - 수정 신고 필요 여부 확인
```

### 🔹 **2단계: 이전 실행 시 준비 사항**

| 절차 | 기한 | 제출처 | 비고 |
|------|------|--------|------|
| 본점이전 등기 | 이전일로부터 2주 내 | 법원 | 등기필증 확보 |
| 사업자등록 정정 | 이전일로부터 10일 내 | 성동세무서 | 업종코드 변경 금지 |
| 지방세 신고 | 폐업/개업 신고 | 청주시청→서울시청 | 법인지방소득세 납세지 변경 |
| 창업세액감면 계속 신청 | 다음 사업연도 법인세 신고 시 | 성동세무서 | 별도 신청서 불필요 (자동 승계) |

```
월별 체크리스트:
```

## 🎯 최적 솔루션 제안

### **[권장 전략] 단계적 이전 + 세무 리스크 헤지**

#### **Plan A. 안전형: 지점 방식**
```
청주 본점 유지 + 서울 지점 설치
→ 감면 100% 유지하며 채용 진행
→ 감면 종료 후 본점 이전
```
**장점**: 세무 리스크 0%  
**단점**: 법인세 신고 복잡도 증가, 임대료 이중 부담

#### **Plan B. 공격형: 즉시 이전** ✅ **추천**
```
업종코드 'C72201' 확인 후 즉시 이전
→ 지식기반산업 인정 시 감면 지속
→ 우수 인재 조기 확보로 매출 성장
```
**전제조건**: 관할 세무서 사전 질의서 제출 (회신 2주 소요)
