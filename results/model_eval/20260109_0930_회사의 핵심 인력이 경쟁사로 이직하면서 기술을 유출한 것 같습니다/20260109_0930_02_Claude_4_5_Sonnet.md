- 경쟁사의 최근 제품 출시 동향
- 재직 중 특허·직무발명 등록 여부

## ⚖️ 승소 가능성 분석 (법률 전문가 관점)

### ✅ **유리한 요소들**
1. **업종 특성**: 양자암호 기술은 고도의 전문성이 요구되는 영역으로, 법원이 영업비밀 보호 필요성을 높게 평가
2. **객관적 증거**: 소스코드 대량 다운로드 로그는 강력한 물적 증거
3. **고객군의 특수성**: 금융·국방 분야는 기술 유출 시 공익적 피해가 크므로 법원의 엄격한 판단 가능

### ⚠️ **불리한 요소들**
1. **경업금지 약정의 유효성 문제**
   - 대법원 판례상 다음 조건 미충족 시 무효 위험:
     - ① **보상의 적정성**: 퇴직금 외 별도 금전 보상 제공 여부
     - ② **기간·지역의 합리성**: 통상 2년 이하, 특정 지역/업종으로 한정
     - ③ **근로자의 생계 보장**: 과도하게 전직을 제한하는지 여부

2. **"영업비밀"의 입증 난이도**
   - 부정경쟁방지법상 보호받으려면:
     - ① 비공지성(외부에 알려지지 않음)
     - ② 경제적 유용성
     - ③ **비밀관리성**(접근 통제, 보안 교육, NDA 체결 등)
   - ⚠️ **귀사의 리스크**: 소스코드에 대한 체계적 접근 통제가 없었다면 입증 약화

3. **손해액 산정의 어려움**
   - 실제 손해 발생을 금액으로 특정하기 어려워 법원이 낮게 인정할 가능성

## 🎯 Step-by-Step 실행 가이드

### **STEP 1: 긴급 증거 확보 (당장 이번 주 내)**
```
✓ 디지털 포렌식 전문 업체에 의뢰 (추천: KISA 한국인터넷진흥원 연계 기관)
  - 퇴직자의 PC, 이메일, 클라우드 접속 로그 전수 조사
  - 외장하드, USB 사용 이력 확인

✓ 내부 영업비밀 관리 체계 재점검
  - 기존 보안 정책 문서화 (소급 적용 X, 향후 증거로 활용)
  - 재직 중 체결한 NDA, 보안서약서 원본 확보

✓ 경쟁사 모니터링
  - 경쟁사의 신제품 발표, 채용 공고, 특허 출원 등 추적
  - 동종 업계 인맥 통해 정보 수집 (합법적 범위 내)
```

### **STEP 2: 법적 경고 및 가처분 신청 (2주차)**
```
✓ 변호사 선임 (필수)
  - 영업비밀 소송 경험이 있는 법무법인 (예: 로앤굿, 화우 IP팀 등)
  - 초기 자문 시 증거 자료 전체 공유

✓ 내용증명 발송
  - 대상: 퇴직자 + 경쟁사 대표이사
  - 내용: 영업비밀 침해 중단, 자료 반환, 손해배상 예고

✓ 가처분 신청 검토
  - "퇴직자의 경쟁사 근무 금지" 가처분
  - 승인 시 즉각적 효과, 단 보증금(수천만 원) 필요
```

### **STEP 3: 본안 소송 준비 (1개월차~)**
```
✓ 민사: 손해배상 청구
  - 청구 금액: 개발 비용 + 예상 매출 손실 (보수적 산정 권장)
  - 소송비용: 인지대 + 변호사 비용 최소 5,000만 원 이상 예상

✓ 형사: 영업비밀 침해 고소 (선택적)
  - 부정경쟁방지법 위반 (3년 이하 징역 또는 3억 원 이하 벌금)
  - 장점: 검찰 수사를 통한 강제 증거 확보
  - 단점: 입증 기준이 높고, 형사처벌이 손해배상을 보장하지 않음
```

## 💡 승소 가능성 종합 평가

### **현재 정보 기준 예측**
- **경업금지 위반 청구**: **40~60%** (약정 유효성 여부에 따라 변동 큰 폭)
- **영업비밀 침해 청구**: **50~70%** (포렌식 결과에 따라 상승 가능)
- **손해배상 인용 금액**: 청구액의 **20~40%** 수준 (업계 평균)

### **승소 가능성을 높이는 핵심 3가지**
1. **디지털 포렌식 결과의 명확성** (직접 증거 확보)
2. **경업금지 약정의 합리성** (보상금 + 기간 제한)
3. **영업비밀 관리 체계의 입증** (문서화된 보안 정책)
